# 알림 정책

import { Callout } from 'nextra/components'

알림 시스템은 단순한 공지 수단이 아니라, 재등록률·이탈 방지·수납률에 직접적인 영향을 미치는 비즈니스 도구입니다. 만료 전 알림은 회원이 자연스럽게 재등록을 결정하도록 유도하고, 미출석 알림은 잠재적 이탈 회원을 조기에 발견해 관리할 수 있게 합니다. 미수금 독촉은 수납률을 높여 센터의 현금 흐름을 안정시킵니다. 잘 설계된 알림 시스템은 직원의 수작업을 줄이면서도 회원 관리 품질을 높이는 운영 효율화 수단입니다.

## 만료 사전 알림

만료 알림의 시점은 심리적 긴박감을 단계적으로 높이도록 설계되어 있습니다. 30일 전에는 아직 여유가 있으므로 앱 푸시로 가볍게 안내합니다. 14일 전부터는 SMS를 추가해 인지도를 높입니다. 7일·3일·당일로 갈수록 발송 빈도가 높아지고, 만료 후 3일에는 재등록 유도 메시지를 발송합니다. 이 시점 설계는 "아직 시간 있다 → 슬슬 생각해야겠다 → 이번 주 안에 결정해야 한다 → 오늘이 마지막"이라는 심리적 전환을 자연스럽게 유도합니다.

| 시점 | 채널 |
|---|---|
| 30일 전 | 앱 푸시 |
| 14일 전 | SMS + 앱 푸시 |
| 7일 전 | SMS |
| 3일 전 | SMS |
| 당일 | SMS |
| 만료 후 3일 | SMS (재등록 유도) |

- 발송: 09:00~21:00
- 채널 우선순위: 앱 푸시(무료) → 카카오 → SMS(유료)

## 미출석 알림

미출석 알림은 단계별 에스컬레이션 구조로 설계되어 있습니다. 초기에는 회원에게 직접 알림을 보내 자발적 복귀를 유도하고, 이후에는 담당 FC·트레이너가 개입하여 개인적인 연락을 취합니다. 21일 이상 미출석은 전화 상담을 필수로 지정하여, 단순 메시지가 아닌 인간적 관계로 이탈을 방지합니다. 60일 이상이 되면 본사 차원에서 이탈 위험 리포트를 생성하여 패턴 분석과 대응 전략 수립에 활용합니다.

| 기간 | 대상 | 행동 |
|---|---|---|
| 7일 | 회원 | 앱 푸시 |
| 14일 | 회원 + FC/트레이너 | SMS + 시스템 |
| 21일 | 회원 + FC/트레이너 | 전화 상담 필수 |
| 30일 | 지점장 | 특별 혜택 검토 |
| 60일 | 본사 | 이탈 위험 리포트 |

## 미수금 독촉

<Callout type="warning">
채권추심에 관한 법률에 따라 독촉 메시지 발송은 **09:00~20:00** 사이에만 허용됩니다. 야간(20:00~09:00) 및 공휴일 발송은 법적으로 금지되어 있습니다. 시스템은 이 시간 외 발송을 자동으로 차단합니다.
</Callout>

미수금 독촉 메시지는 기간에 따라 톤이 달라집니다. 초기에는 안내 성격으로 시작하여, 기간이 길어질수록 점차 강도를 높입니다. 30일을 넘기면 회원 상태가 SUSPENDED로 전환되어 출입이 차단됩니다. 이는 미납 상태의 회원이 시설을 무제한 이용하는 상황을 방지하기 위한 조치입니다.

| 미납 기간 | 톤 |
|---|---|
| 3일 | 안내 |
| 7일 | 안내 (금액, 납부 방법) |
| 14일 | 독촉 + 지점장/본사 알림 |
| 30일 | 경고 + SUSPENDED |

- 발송: 09:00~20:00, 일 1회, 야간/휴일 금지

## 생일/이벤트

생일 쿠폰, 등록 기념일 혜택, 출석 마일리지는 회원의 소속감과 만족도를 높이는 감성 마케팅 수단입니다. 자동화된 이벤트 알림은 직원의 수작업 없이도 회원 개개인에게 맞춤형 경험을 제공하여, 재등록률과 장기 회원 비율을 높이는 데 기여합니다.

| 이벤트 | 시점 | 내용 |
|---|---|---|
| 생일 | 당일 09:00 | 축하 + 쿠폰 |
| 등록 1주년 | 해당일 | 기념 혜택 |
| 100회 출석 | 달성 즉시 | 마일리지 |

## 내부 알림 (직원 간)

내부 알림은 지점 내·본사 간 실시간 정보 공유를 통해 운영 효율을 높입니다. 예를 들어 50만 원 이상 결제나 환불이 발생하면 지점장과 본사가 즉시 알림을 받아 비정상 거래를 빠르게 감지할 수 있습니다. 수기 매출 등록 알림은 임의 매출 조작을 억제하는 내부 통제 수단이기도 합니다.

| 이벤트 | 수신 |
|---|---|
| 환불 요청 | 지점장 + 본사(50만+) |
| 50만 원+ 결제 | 지점장 + 본사 |
| 수기 매출 등록 | 본사(대표) |
| 수업 취소 (강사) | 예약 회원 + 지점장 |
| 직원 지각/결근 | 지점장 |
| 블랙리스트 출입 | 지점장 + 본사 |
| 결재 요청 | 본사(대표/회장) |
| 결재 승인/반려 | 지점장 |
| 급여 확정 | 전 직원(본인) |

## 메시지 발송

메시지 발송 기능을 통해 특정 회원 또는 그룹에게 직접 메시지를 전송할 수 있습니다. 자동 알림과 달리 수동으로 내용을 작성하여 발송합니다.

### 발송 절차

메시지 메뉴 → "새 메시지" → 수신자 선택 → 내용 작성 → 발송

- **수신자 선택**: 개별 회원, 그룹(태그/조건), 전체 회원 중 선택
- **메시지 유형**: SMS(90자), LMS(2000자), 카카오톡(1000자) 중 선택
- **변수 치환**: `#{이름}`, `#{days_left}`, `#{membership_end}` 등 자동 치환 지원
- **예약 발송**: 발송 일시를 지정하여 원하는 시간에 발송 가능
- **발송 이력**: 발송 목록에서 성공/실패 건수 확인 가능

### 자동 알림 설정

설정 → 자동 알림 메뉴에서 규칙별 알림을 관리합니다.

- 규칙별 ON/OFF 전환
- 발송 시간 설정
- 메시지 템플릿 수정

### 발송 비용 확인

메시지 → 발송 통계 메뉴에서 채널별 발송 건수와 비용을 확인할 수 있습니다.

## 비용 관리

SMS는 건당 비용이 발생하는 유료 채널입니다. 앱 푸시와 카카오 알림톡 대비 단가가 높으므로, 가능한 경우 무료 채널을 우선 활용하고 SMS는 최후 수단으로 사용합니다. 대량 발송(100건 이상)은 단기간에 높은 비용이 발생할 수 있어 본사 결재를 통해 예산 통제를 합니다. 지점별 SMS 한도를 설정하면 지점이 예산 범위 내에서 자율적으로 운영할 수 있습니다.

| 항목 | 규칙 |
|---|---|
| SMS 한도 | 본사 설정 (지점별) |
| 한도 초과 | 본사 승인 |
| 대량 발송 (100건+) | 본사 결재 |
| 리포트 | 월간 채널별 비용 (본사+) |
