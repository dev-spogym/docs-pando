## 로그인

CRM 로그인은 이메일과 비밀번호를 입력한 뒤 본인 소속 지점을 선택하는 2단계로 구성됩니다. 본사 역할(회장/대표/본사직원)은 로그인 후 언제든지 전 지점을 자유롭게 전환할 수 있습니다.

- **로그인 화면**: 이메일 + 비밀번호 입력 → 지점 선택 → 로그인
- **지점 선택**: 로그인 후 본인 소속 지점 선택 (본사 역할은 전 지점 전환 가능)
- **비밀번호 찾기**: 로그인 화면 → "비밀번호 찾기" 링크 → 이메일 인증 → 재설정
- **비밀번호 변경 알림**: 90일 경과 시 로그인 후 변경 안내 화면 자동 표시
- **동시 세션**: 다른 곳에서 로그인 시 기존 세션 자동 종료 + 알림

---

피트니스 CRM은 세 가지 유형의 민감 정보를 동시에 다룹니다. 첫째, 회원의 이름·연락처·생년월일 등 개인정보, 둘째, 카드번호·계좌번호·결제 내역 등 금융정보, 셋째, 체성분·건강 상태·진단서 등 건강정보입니다. 이 중 하나라도 유출되면 개인정보보호법 위반에 따른 과징금, 피해자 손해배상 청구, 브랜드 신뢰 훼손이 발생합니다. 또한 직원 계정이 도용되면 회원 데이터 열람, 허위 결제, 급여 조작 등 내부 부정 행위로도 악용될 수 있습니다.

## 비밀번호

비밀번호 정책은 OWASP(Open Web Application Security Project)의 인증 보안 가이드라인을 기반으로 설계되었습니다. OWASP는 8자 이상, 문자 조합 다양성, 주기적 변경, 재사용 방지를 핵심 권고 사항으로 제시합니다. 특히 초기 비밀번호를 최초 로그인 시 반드시 변경하도록 강제하는 것은, 관리자가 설정한 임시 비밀번호가 공유·노출될 위험을 차단하기 위함입니다.

<Callout type="warning">
비밀번호 5회 오류 시 30분 계정 잠금, 10회 오류 시 영구 잠금이 적용됩니다. 영구 잠금 해제는 지점장(직원 계정) 또는 본사(지점장 계정)만 가능합니다.
</Callout>

| 항목 | 규칙 |
|---|---|
| 최소 길이 | 8자 |
| 복잡도 | 영문+숫자+특수문자 3종 조합 |
| 변경 주기 | 90일 권고, 180일 강제 |
| 재사용 금지 | 최근 3개 |
| 초기 비밀번호 | 최초 로그인 시 강제 변경 |
| 오류 | 5회→30분 잠금, 10회→영구 |

## 세션

세션 관리는 실제 보안 위협을 직접 차단하는 수단입니다. 대표적인 위협 시나리오는 다음과 같습니다.

- **좌석 이탈 공격:** 직원이 로그인한 채 자리를 비운 사이, 다른 사람이 해당 컴퓨터에 접근해 회원 정보를 열람하거나 결제를 조작하는 경우. 비활성 30분 타임아웃이 이를 방지합니다.
- **장기 세션 탈취:** 쿠키나 토큰이 탈취되어 오랫동안 악용되는 경우. 절대 타임아웃 12시간이 이를 제한합니다.
- **다중 기기 공유:** 동일 계정을 여러 직원이 공유하면 행위 추적이 불가능해져 내부 통제가 무력화됩니다. 1계정 1기기 정책이 이를 방지합니다.

| 항목 | 규칙 |
|---|---|
| 비활성 타임아웃 | 30분 |
| 절대 타임아웃 | 12시간 |
| 동시 로그인 | 1계정 1기기 |
| 강제 로그아웃 | 지점장+(직원), 본사(지점장) |

## IP/기기 접근 제한

IP 화이트리스트는 지정된 IP 대역(예: 지점 내부망)에서만 시스템 접근을 허용하는 기능입니다. 기본은 OFF이지만, 보안 요구가 높은 본사 또는 특정 지점에서 활성화할 수 있습니다. 외부 2FA(2단계 인증)는 모바일 OTP 등을 통해 비밀번호 외의 추가 인증을 요구합니다.

| 항목 | 규칙 |
|---|---|
| IP 화이트리스트 | 본사 설정 (기본 OFF) |
| 외부 2FA | 본사 설정 |
| 미인가 접근 | 지점장 + 본사 알림 |

## 접근 로그 모니터링

시스템은 접근 패턴을 실시간으로 분석하여 비정상 행위를 탐지합니다. 정상적인 업무라면 심야에 CRM에 접속하거나, 짧은 시간에 대량의 데이터를 내려받거나, 반복적으로 삭제를 수행할 이유가 거의 없습니다. 이런 패턴이 감지되면 즉시 담당자에게 알림을 발송하여 빠른 대응이 가능하게 합니다.

**비정상 패턴 예시:**
- 심야(23시~06시)에 회원 목록 조회 → 개인정보 열람 목적의 비인가 접근 의심
- 1시간 내 3회 이상 엑셀 다운로드 → 대량 데이터 유출 시도 의심
- 1시간 내 5회 이상 삭제 → 데이터 조작·증거 인멸 의심
- 로그인 5회 연속 실패 → 계정 탈취 시도(무차별 대입 공격) 의심

| 패턴 | 대응 |
|---|---|
| 심야 접근 (23~06시) | 지점장 + 본사 알림 |
| 대량 다운로드 (1시간 3회+) | 본사 알림 |
| 빈번 삭제 (1시간 5회+) | 본사 알림 + 차단 |
| 로그인 5회 실패 | 잠금 + 본사 알림 |
