# 본사 결재 프로세스

import { Callout } from 'nextra/components'

<Callout type="info">
지점에서 경영/재무/정책에 영향을 미치는 행위를 하려면, **지점장이 본사에 결재를 올리고 승인을 받아야** 해당 행위를 수행할 수 있습니다.
</Callout>

결재 프로세스는 지점의 운영 자율성과 본사의 경영 통제 사이에서 균형을 잡기 위해 도입되었습니다. 지점장이 매일 발생하는 회원 응대, 수업 운영, 소액 결제 등은 자유롭게 처리할 수 있어야 하지만, 가격 변경이나 대규모 환불처럼 전사 수익성에 영향을 주는 결정은 본사가 사전에 검토하고 승인해야 합니다. 이 체계가 없다면 각 지점이 제각각의 가격 정책과 할인 기준을 적용하여 브랜드 일관성과 수익 관리가 어려워집니다.

## 결재 흐름

결재 요청서에는 행위의 종류, 배경과 사유, 예상 효과, 금액(해당되는 경우), 첨부 서류를 포함해야 합니다. 본사 직원은 요청서를 검토하여 의견을 첨부할 수 있지만, 최종 승인·반려 권한은 대표 또는 회장에게만 있습니다.

```
[지점장: 결재 요청 작성]
    ↓
[본사 직원: 검토 + 의견 첨부] (선택)
    ↓
[대표 또는 회장: 승인/반려/보류]
    ↓
  승인 → 지점에서 해당 행위 수행 가능
  반려 → 사유 첨부 → 지점장에게 반환
  보류 → 추가 자료 요청
```

## 결재 필요 항목

### 할인/프로모션

할인과 프로모션은 지점의 단기 매출에는 도움이 될 수 있지만, 무분별하게 적용되면 브랜드 가치를 훼손하고 전사 마진을 저하시킬 수 있습니다. 특히 10%를 초과하는 할인이나 제휴 계약은 반드시 본사 검토가 필요합니다.

| 행위 | 요청자 | 승인자 |
|---|---|---|
| 할인 쿠폰 생성 | 지점장 | 대표/회장 |
| 프로모션 신규 등록 | 지점장 | 대표/회장 |
| 10% 초과 할인 적용 | 지점장 | 대표/회장 |
| 제휴 업체 할인 계약 | 지점장 | 대표/회장 |

### 상품/가격

상품과 가격은 경영의 핵심 변수입니다. 지점이 임의로 상품을 추가하거나 가격을 바꾸면 지점 간 형평성 문제가 발생하고, 본사의 수익 예측이 어려워집니다. 모든 상품·가격 변경은 본사 결재 후에만 시행할 수 있습니다.

| 행위 | 요청자 | 승인자 |
|---|---|---|
| 신규 상품 등록 | 지점장 | 대표/회장 |
| 상품 가격 변경 | 지점장 | 대표/회장 |
| 상품 폐지/비활성화 | 지점장 | 대표/회장 |

### 환불

소액 환불(50만 원 이하)은 지점장이 고객 응대 차원에서 신속하게 처리할 수 있도록 허용합니다. 그러나 고액 환불이나 위약금 면제는 재무 영향이 크기 때문에 본사 검토가 필요합니다. 위약금 면제는 특히 선례가 되어 다른 회원들의 요구로 이어질 수 있어 신중하게 판단해야 합니다.

| 행위 | 요청자 | 승인자 |
|---|---|---|
| 50만 원 초과 환불 | 지점장 | 대표/회장 |
| 위약금 면제 환불 | 지점장 | 대표/회장 |

### 인사

정규직 채용과 해고는 노동법적 리스크를 수반하는 중요한 의사결정입니다. 급여와 인센티브율 변경은 인건비 구조 전반에 영향을 주므로 특히 급여 체계 변경은 회장 승인을 필요로 합니다.

| 행위 | 요청자 | 승인자 |
|---|---|---|
| 정규직 채용 | 지점장 | 대표/회장 |
| 직원 해고/퇴사 | 지점장 | 대표/회장 |
| 급여 체계 변경 | 지점장 | 회장 |
| 인센티브율 변경 | 지점장 | 대표/회장 |

### 시설/운영/재무

대규모 지출이나 운영 중단은 회사 자산과 고객 신뢰에 직접적인 영향을 줍니다. 특히 지점 일시 중단과 월 마감 확정 취소는 가장 영향 범위가 큰 항목으로 회장 승인이 필요합니다.

| 행위 | 요청자 | 승인자 |
|---|---|---|
| 대형 장비 구매 (100만원+) | 지점장 | 대표/회장 |
| 영업 시간 변경 | 지점장 | 대표/회장 |
| 지점 일시 중단 | 지점장 | 회장 |
| 미수금 탕감 | 지점장 | 대표/회장 |
| 매출 마감 후 수정 | 지점장 | 대표/회장 |
| 월 마감 확정 취소 | 지점장 | 회장 |

## 결재 상태

| 상태 | 설명 |
|---|---|
| `DRAFT` | 작성 중 (지점장) |
| `SUBMITTED` | 제출됨 (본사 검토 대기) |
| `REVIEWING` | 본사 직원 검토 중 |
| `APPROVED` | 승인 → 수행 가능 |
| `REJECTED` | 반려 (사유 첨부) |
| `ON_HOLD` | 보류 (추가 자료 필요) |

## 결재 규칙

<Callout type="warning">
승인된 결재는 **승인일로부터 30일** 이내에 이행해야 합니다. 기간 내 미이행 시 결재 효력이 소멸되어 재요청이 필요합니다.
</Callout>

긴급 상황(예: 헬스장 장비 고장으로 당일 새 장비 구매 필요)에는 대표 또는 회장의 전화 승인을 먼저 받고 24시간 내에 사후 결재를 처리할 수 있습니다. 모든 결재 이력은 AuditLog에 영구 보존됩니다.

| 항목 | 규칙 |
|---|---|
| 요청 권한 | 지점장만 |
| 긴급 결재 | 대표/회장 전화 승인 → 24시간 내 사후 처리 |
| 유효 기간 | 승인일로부터 30일 |
| 본사 직원 | 검토 + 의견만 (승인/반려 불가) |

## 결재 화면 사용법

### 결재 요청 작성

결재 메뉴에서 "새 결재" 버튼을 누릅니다. 결재 유형을 선택한 후 내용, 사유, 금액(해당하는 경우)을 입력하고 제출합니다. 제출하면 상태가 `SUBMITTED`로 변경되어 본사에서 확인할 수 있습니다.

### 결재 현황 확인

결재 메뉴에서 상태별 필터(대기중 / 승인 / 반려 / 보류)를 선택하면 해당 상태의 결재 목록만 표시됩니다. 각 결재 항목을 클릭하면 본사 의견과 처리 이력을 확인할 수 있습니다.

### 결재 처리 (본사 역할)

본사 역할로 로그인하면 결재 목록에서 제출된 결재 요청을 확인할 수 있습니다. 요청 내용을 검토한 후 "승인", "반려", "보류" 중 하나를 선택합니다. 반려 또는 보류 시에는 사유 또는 추가 요청 사항을 반드시 입력해야 합니다.

### 긴급 결재

긴급 상황에서 전화로 대표 또는 회장의 구두 승인을 받은 경우, 24시간 이내에 결재 메뉴에서 해당 내용을 사후 등록해야 합니다. 긴급 결재 유형을 선택하고 전화 승인 일시와 승인자를 기록하세요.

## 결재 없이 지점장 단독 수행 가능

일상적인 지점 운영에 필요한 항목들은 결재 없이 지점장이 직접 처리할 수 있습니다. 이 항목들은 매일 반복적으로 발생하며 본사 개입 없이 즉시 처리해야 하는 업무입니다. 결재가 필요한지 불확실한 경우, 금액이 크거나 전례가 없는 상황이라면 사전에 본사와 협의하는 것을 권장합니다.

| 카테고리 | 행위 |
|---|---|
| 회원 | 등록/수정/탈퇴, 휴회/정지, 이관 요청 |
| 수업 | 등록/수정/삭제, 시간표, 페널티 |
| 매출 | POS 결제, 50만 원 이하 환불, 미수금 수납 |
| 시설 | 락커/RFID/운동룸/운동복 일상 관리 |
| 메시지 | 100건 미만 발송, 기 승인 쿠폰 발급 |
| 직원 | 근태 관리, 파트타임 채용 |
