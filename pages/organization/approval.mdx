# 본사 결재 프로세스

import { Callout } from 'nextra/components'

<Callout type="info">
지점에서 경영/재무/정책에 영향을 미치는 행위를 하려면, **지점장이 본사에 결재를 올리고 승인을 받아야** 해당 행위를 수행할 수 있습니다.
</Callout>

## 결재 흐름

```
[지점장: 결재 요청 작성]
    ↓
[본사 직원: 검토 + 의견 첨부] (선택)
    ↓
[대표 또는 회장: 승인/반려/보류]
    ↓
  승인 → 지점에서 해당 행위 수행 가능
  반려 → 사유 첨부 → 지점장에게 반환
  보류 → 추가 자료 요청
```

## 결재 필요 항목

### 할인/프로모션

| 행위 | 요청자 | 승인자 |
|---|---|---|
| 할인 쿠폰 생성 | 지점장 | 대표/회장 |
| 프로모션 신규 등록 | 지점장 | 대표/회장 |
| 10% 초과 할인 적용 | 지점장 | 대표/회장 |
| 제휴 업체 할인 계약 | 지점장 | 대표/회장 |

### 상품/가격

| 행위 | 요청자 | 승인자 |
|---|---|---|
| 신규 상품 등록 | 지점장 | 대표/회장 |
| 상품 가격 변경 | 지점장 | 대표/회장 |
| 상품 폐지/비활성화 | 지점장 | 대표/회장 |

### 환불

| 행위 | 요청자 | 승인자 |
|---|---|---|
| 50만 원 초과 환불 | 지점장 | 대표/회장 |
| 위약금 면제 환불 | 지점장 | 대표/회장 |

### 인사

|---|---|---|
| 정규직 채용 | 지점장 | 대표/회장 |
| 직원 해고/퇴사 | 지점장 | 대표/회장 |
| 급여 체계 변경 | 지점장 | 회장 |

## 결재 상태

| 상태 | 설명 |
|---|---|
| `DRAFT` | 작성 중 (지점장) |
| `SUBMITTED` | 제출됨 (본사 검토 대기) |
| `REVIEWING` | 본사 직원 검토 중 |
| `APPROVED` | 승인 → 수행 가능 |
| `REJECTED` | 반려 (사유 첨부) |
| `ON_HOLD` | 보류 (추가 자료 필요) |

## 결재 규칙

| 항목 | 규칙 |
|---|---|
| 요청 권한 | 지점장만 |
| 긴급 결재 | 대표/회장 전화 승인 → 24시간 내 사후 처리 |
| 유효 기간 | 승인일로부터 30일 |
| 본사 직원 | 검토 + 의견만 (승인/반려 불가) |

## 결재 없이 지점장 단독 수행 가능

| 카테고리 | 행위 |
|---|---|
| 회원 | 등록/수정/탈퇴, 휴회/정지, 이관 요청 |
| 수업 | 등록/수정/삭제, 시간표, 페널티 |
| 매출 | POS 결제, 50만 원 이하 환불, 미수금 수납 |
| 시설 | 락커/RFID/운동룸/운동복 일상 관리 |
| 메시지 | 100건 미만 발송, 기 승인 쿠폰 발급 |
| 직원 | 근태 관리, 파트타임 채용 |
