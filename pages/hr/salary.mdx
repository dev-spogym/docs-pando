# 급여 체계 / 인센티브

## 급여 유형

| 유형 | 대상 | 구성 |
|---|---|---|
| 월급제 | 지점장, 스태프 | 기본급 + 직책수당 + 인센티브 - 공제 |
| 시급제 | 파트타임 | 시급 × 근무시간 + 인센티브 - 공제 |
| 건별제 | 트레이너 (프리랜서) | 건당 단가 + 인센티브 - 공제 |
| 혼합제 | FC, 트레이너 (정규) | 기본급 + 성과 수당 + 인센티브 - 공제 |

급여 체계 결정: **본사(회장/대표)**, 변경 시 본사 결재

## PT 수업 인센티브 (트레이너)

```
PT 인센티브 = PT 매출 × 인센티브율

인센티브율:
  월 300만 원 이하 → 30%
  월 300~500만 원 → 35%
  월 500~800만 원 → 40%
  월 800만 원 초과 → 45%
```

## GX 수업 인센티브 (트레이너)

```
GX 인센티브 = 수업 횟수 × 회당 단가

회당 단가:
  5인 이하 → 20,000원
  6~10인 → 25,000원
  11~15인 → 30,000원
  16인+ → 35,000원
```

## FC 인센티브

```
판매 인센티브 = 본인 판매 매출 × 수수료율

수수료율:
  이용권 판매 → 8~15%
  재등록 유도 → 건당 20,000원
  신규 유치 (상담→계약) → 건당 30,000원
```

## 인센티브 관리

| 행위 | 권한 |
|---|---|
| 구간/율 설정 | 본사, 지점장 → 결재 |
| 전 직원 조회 | 본사, 지점장 |
| 본인 조회 | 전 역할 |
| 정산 주기 | 매월 (익월 급여 합산) |
