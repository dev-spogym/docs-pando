# 정산 / 퇴직금

import { Callout } from 'nextra/components'

급여 정산은 직원의 고용 형태에 따라 세금 처리 방식이 완전히 달라집니다. 정규직 근로자는 4대보험 가입과 근로소득세 원천징수가 의무이며, 프리랜서(사업소득자)는 3.3% 원천징수로 간소화됩니다. 잘못된 처리는 가산세와 4대보험 미납 처벌로 이어질 수 있으므로 고용 형태 분류가 중요합니다.

## 4대보험/세금

### 정규직

정규직 직원은 국민연금, 건강보험, 고용보험을 근로자와 사업자가 각각 절반씩 부담합니다. 소득세는 국세청 간이세액표를 기준으로 공제하며, 연말정산을 통해 최종 세액을 확정합니다.

<Callout type="warning">
4대보험은 직원 채용 즉시 신고 의무가 발생합니다. 수습 기간 중이더라도 동일하게 적용됩니다. 미신고 시 과태료가 부과됩니다.
</Callout>

| 항목 | 근로자 | 사업자 |
|---|---|---|
| 국민연금 | 4.5% | 4.5% |
| 건강보험 | 3.545% | 3.545% |
| 장기요양 | 건강보험의 12.81% | 동일 |
| 고용보험 | 0.9% | 0.9~1.65% |
| 소득세 | 간이세액표 | - |

### 프리랜서

프리랜서(건별제 트레이너 등)는 근로계약이 아닌 용역 계약 관계이므로 4대보험이 적용되지 않습니다. 대신 지급액의 3.3%(소득세 3% + 지방소득세 0.3%)를 원천징수하여 다음 달 10일까지 세무서에 신고·납부해야 합니다. 프리랜서는 5월 종합소득세 신고를 통해 연간 세액을 정산합니다.

**정규직과 프리랜서 세금 처리 차이 요약:**
- 정규직: 4대보험 가입 의무 + 근로소득 원천징수 + 연말정산
- 프리랜서: 4대보험 없음 + 사업소득 3.3% 원천징수 + 종합소득세 신고

소득세+지방소득세 = **3.3%** 원천징수

## 급여 마감 프로세스

급여 마감은 5단계를 거쳐 최종 지급까지 약 10일이 소요됩니다. 각 단계의 책임과 역할은 명확하게 구분됩니다.

- **1단계 - 자동 산출 (1일):** 시스템이 근태 기록, 인센티브 실적, 공제 항목을 자동으로 계산하여 DRAFT 상태로 생성합니다.
- **2단계 - 지점장 검토 (1~3일):** 지점장이 근태 이상, 인센티브 누락, 특별 수당 등을 확인하고 이상이 없으면 PENDING으로 전환합니다.
- **3단계 - 본사 검토/수정 (3~5일):** 본사(hq_staff)가 전 지점 급여를 검토하고 수정합니다. 급여 체계 적정성, 인센티브율 적용 여부 등을 확인합니다.
- **4단계 - 대표/회장 확정 (5일):** 최종 급여를 CONFIRMED 상태로 확정합니다. 이 시점 이후 수정은 회장만 가능합니다.
- **5단계 - 지급 및 명세서 (10일):** 은행 이체를 통해 지급하고, 급여 명세서가 자동으로 생성되어 직원에게 발송됩니다.

```
[자동 산출] → [지점장 검토] → [본사 검토] → [대표/회장 확정] → [지급] → [명세서]
  1일(시스템)   1~3일         3~5일(본사)     5일             10일    10일(자동)
```

| 상태 | 수정 권한 |
|---|---|
| DRAFT | 본사 자유 수정 |
| PENDING | 본사 수정 가능 |
| CONFIRMED | 회장만 (사유 필수) |
| PAID | 수정 불가 |

**급여 확정: 본사(대표/회장)만 가능**

## 퇴직금 산정

퇴직금은 근로자퇴직급여보장법에 따라 1년 이상 근속한 모든 직원에게 지급 의무가 있습니다. 퇴직일로부터 14일 이내에 지급해야 하며, 지연 시 지연이자(연 20%)가 발생합니다.

<Callout type="warning">
**근로자퇴직급여보장법:** 1년 이상 근속 직원에게 퇴직금 미지급 시 형사처벌(3년 이하 징역 또는 3천만 원 이하 벌금) 대상입니다. 퇴직 후 14일 이내 지급이 의무입니다.
</Callout>

**퇴직금 산정 예시:**
- 기본 정보: 월 기본급 300만 원, 정기 인센티브 월 평균 50만 원, 총 근속 2년 6개월(912일)
- 퇴직 전 3개월 총 급여: (300만 + 50만) × 3개월 = 1,050만 원
- 해당 기간 일수: 91일 (예: 1월~3월)
- 1일 평균임금: 10,500,000원 ÷ 91일 = 115,385원
- 퇴직금: 115,385원 × 30일 × (912일 ÷ 365일) = 115,385원 × 30 × 2.497 = **약 8,647,000원**

```
퇴직금 = 1일 평균임금 × 30일 × (총 근속일수 ÷ 365)
1일 평균임금 = 퇴직 전 3개월 총 급여 ÷ 해당 기간 총 일수
```

| 항목 | 규칙 |
|---|---|
| 대상 | 1년 이상 근속 |
| 인센티브 | 정기적 인센티브는 평균임금에 포함 |
| 지급 기한 | 퇴직일로부터 14일 이내 |
| 승인 | 대표/회장 |

## 카드 수수료 정산

카드 수수료는 회원이 카드로 결제할 때 센터가 카드사에 지급하는 수수료입니다. 예를 들어 100만 원 이용권을 카드로 결제하면 1.5~2.5%인 1만 5천~2만 5천 원을 카드사에 지불하게 됩니다. 이는 실제 수입에서 차감되므로, 매출 통계에서 총매출·수수료·순매출을 구분하여 표시해 경영 판단에 활용합니다.

**카드 수수료가 매출에 미치는 영향:**
월 총 카드 매출이 3,000만 원이고 평균 수수료율이 2%라면, 수수료로 60만 원이 빠져나가 순매출은 2,940만 원이 됩니다. 연간으로는 720만 원의 차이가 발생하므로, 카드 수수료는 무시할 수 없는 비용 항목입니다. 카드 결제 비중이 높은 센터일수록 수수료 절감을 위해 카드사와의 협약 수수료율 협상이 중요합니다. 영세 사업자(연 매출 3억 원 이하)는 우대 수수료율(0.5~1.5%)을 적용받을 수 있으므로 확인이 필요합니다.

| 항목 | 규칙 |
|---|---|
| 수수료율 관리 | 본사 |
| 기본율 | 일반 1.5~2.5%, 영세 0.5~1.5% |
| 순매출 표시 | 총매출/수수료/순매출 구분 |
| 정산 주기 | D+2 |

## CRM에서 급여명세서·정산 확인하기

### 급여 명세서 조회

직원 본인은 CRM 좌측 메뉴에서 **급여 → 급여 명세서**를 선택하고 조회할 월을 선택하면 해당 월 명세서를 확인할 수 있습니다. 명세서에는 기본급, 각종 수당, 인센티브, 공제 항목(4대보험·소득세), 실지급액이 항목별로 상세 표시됩니다.

### PDF 다운로드

명세서 화면 우측 상단의 **PDF 다운로드** 버튼을 클릭하면 회사 정보, 센터명, 직원 정보가 포함된 공식 급여 명세서를 PDF 파일로 저장할 수 있습니다.

<Callout type="info">
급여 명세서는 본인 것만 조회할 수 있습니다. 지점장과 본사는 전 직원 명세서를 조회·다운로드할 수 있습니다.
</Callout>

### 퇴직 정산

직원 퇴사 처리 시 시스템이 근속 기간과 최근 3개월 평균임금을 기준으로 퇴직금을 자동으로 계산합니다. 계산된 퇴직금은 본사 승인 후 지급 처리됩니다. 퇴직금 정산 현황은 해당 직원 상세 화면의 **정산** 탭에서 확인할 수 있습니다.

<Callout type="warning">
퇴직금은 퇴직일로부터 14일 이내에 지급해야 합니다. 지급이 지연되면 지연이자(연 20%)가 발생하므로 퇴사 처리 즉시 정산 절차를 시작하세요.
</Callout>

### 카드 수수료 확인

CRM 매출 통계 화면에서 총매출, 카드 수수료, 순매출을 구분하여 확인할 수 있습니다. 월별 수수료 합계와 비율도 함께 표시되므로, 실제 수익 규모를 정확히 파악하는 데 활용하세요.
