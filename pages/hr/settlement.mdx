## 급여 마감 프로세스

```
[자동 산출] → [지점장 검토] → [본사 검토] → [대표/회장 확정] → [지급] → [명세서]
  1일(시스템)   1~3일         3~5일(본사)     5일             10일    10일(자동)
```

| 상태 | 수정 권한 |
|---|---|
| DRAFT | 본사 자유 수정 |
| PENDING | 본사 수정 가능 |
| CONFIRMED | 회장만 (사유 필수) |
| PAID | 수정 불가 |

**급여 확정: 본사(대표/회장)만 가능**

## 퇴직금 산정

> 법적 근거: 근로자퇴직급여보장법

```
퇴직금 = 1일 평균임금 × 30일 × (총 근속일수 ÷ 365)
1일 평균임금 = 퇴직 전 3개월 총 급여 ÷ 해당 기간 총 일수
```

| 항목 | 규칙 |
|---|---|
| 대상 | 1년 이상 근속 |
| 인센티브 | 정기적 인센티브는 평균임금에 포함 |
| 지급 기한 | 퇴직일로부터 14일 이내 |
| 승인 | 대표/회장 |
