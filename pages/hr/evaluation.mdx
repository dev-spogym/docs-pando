# 수습 / 평가 / 계정잠금 / 휴직

## 수습 기간

| 항목 | 규칙 |
|---|---|
| 기간 | 3개월 |
| 급여 | 기본급 90% (최저임금 이상) |
| 권한 | 정규 직원 동일 |
| 평가 | 종료 2주 전 지점장 → 본사 보고 |
| 전환 | 통과 → 정규직, 미통과 → 본사 결재 후 종료 |

## 직원 평가

| 항목 | 규칙 |
|---|---|
| 주기 | 반기 1회 (6월, 12월) |
| 항목 | 수업 품질(30%), 회원 관리(25%), 매출 기여(25%), 근태(10%), 팀워크(10%) |
| 평가자 | 지점장 → 본사 검토 |
| 등급 | S(탁월) / A(우수) / B(양호) / C(미흡) / D(부적격) |
| 활용 | 인센티브 계수, 연봉 조정 |
| 열람 | 본인 결과만 |
| 이의 | 7일 내 재평가 요청 |

## 계정 잠금

| 조건 | 잠금 |
|---|---|
| 비밀번호 5회 오류 | 30분 |
| 비밀번호 10회 오류 | 영구 |
| 90일 미접속 | 자동 |
| 해제 | 지점장+(지점), 본사(지점장) |

## 휴직

| 사유 | 최대 기간 | 급여 |
|---|---|---|
| 육아 (법정) | 1년 | 고용보험 |
| 병가 | 6개월 | 무급 |
| 개인 사유 | 3개월 | 무급 |

- 담당 재배정: 지점장
- 복직: 7일 전 신청 → 지점장 승인
- 장기 휴직(3개월+): 본사 결재
