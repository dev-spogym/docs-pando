# 수습 / 평가 / 계정잠금 / 휴직

import { Callout } from 'nextra/components'

직원의 입사부터 휴직까지 전 과정에서 법적 요건과 내부 기준을 동시에 준수해야 합니다. 수습 기간은 채용 적합성을 검증하는 시간이며, 정기 평가는 직원의 성장과 공정한 보상의 기준이 됩니다. 계정 보안과 휴직 관리는 각각 정보보안과 노동법의 요구사항을 반영합니다.

## 수습 기간

수습 기간은 근로계약 체결 후 실제 업무 능력과 조직 적합성을 상호 확인하는 법적으로 유효한 기간입니다. 수습 직원도 정규 직원과 동일한 권한을 가지며, 급여만 기본급의 90%로 설정됩니다. 단, 최저임금법에 의해 최저임금 이하로 지급할 수 없으므로 반드시 최저임금 이상임을 확인해야 합니다.

<Callout type="warning">
수습 기간 중 계약 종료는 반드시 **본사 결재**를 받아야 합니다. 지점장 단독으로 수습 직원을 해고할 수 없으며, 무단 해고는 부당해고 분쟁으로 이어질 수 있습니다. 종료 사유를 명확히 문서화하고 본사 승인을 받아야 합니다.
</Callout>

수습 기간 종료 2주 전에 지점장이 평가서를 작성하여 본사에 보고합니다. 평가 결과가 미흡하더라도 즉각 해고가 아닌 본사 결재 후 계약 종료 절차를 따릅니다. 이는 부당해고 분쟁 예방과 일관된 인사 관리를 위한 원칙입니다.

| 항목 | 규칙 |
|---|---|
| 기간 | 3개월 |
| 급여 | 기본급 90% (최저임금 이상) |
| 권한 | 정규 직원 동일 |
| 평가 | 종료 2주 전 지점장 → 본사 보고 |
| 전환 | 통과 → 정규직, 미통과 → 본사 결재 후 종료 |

## 직원 평가

반기 평가는 인센티브 계수 조정과 연봉 협상의 기준이 되므로, 공정하고 일관된 기준으로 실시해야 합니다. 평가 항목별 가중치는 역할의 핵심 가치를 반영합니다. 수업 품질(30%)과 회원 관리(25%)가 높은 비중을 차지하는 것은 피트니스 센터의 본질이 서비스 품질에 있기 때문입니다. 매출 기여(25%)는 조직의 지속 가능성을 위한 현실적 지표이며, 근태(10%)와 팀워크(10%)는 조직 문화 유지를 위한 기본 지표입니다.

직원은 평가 결과를 본인만 열람할 수 있으며, 결과에 이의가 있으면 7일 내 재평가를 요청할 수 있습니다. 이는 평가의 투명성과 직원의 권리를 동시에 보장하는 장치입니다.

| 항목 | 규칙 |
|---|---|
| 주기 | 반기 1회 (6월, 12월) |
| 항목 | 수업 품질(30%), 회원 관리(25%), 매출 기여(25%), 근태(10%), 팀워크(10%) |
| 평가자 | 지점장 → 본사 검토 |
| 등급 | S(탁월) / A(우수) / B(양호) / C(미흡) / D(부적격) |
| 활용 | 인센티브 계수, 연봉 조정 |
| 열람 | 본인 결과만 |
| 이의 | 7일 내 재평가 요청 |

## 계정 잠금

계정 잠금은 무단 접근과 무차별 대입 공격(Brute Force)으로부터 시스템을 보호하는 보안 정책입니다. 비밀번호 5회 오류 시 30분 잠금은 자동화된 공격을 차단하고, 10회 오류 시 영구 잠금은 지속적인 침입 시도를 완전히 차단합니다. 90일 미접속 자동 잠금은 퇴직 직원의 계정이 방치되어 악용되는 것을 방지합니다.

**실무 처리 방법:** 직원이 비밀번호를 분실했거나 계정이 잠겼을 경우, 지점 직원은 지점장에게, 지점장 계정은 본사에 해제를 요청합니다. 해제 전 본인 확인 절차를 반드시 거쳐야 합니다.

| 조건 | 잠금 |
|---|---|
| 비밀번호 5회 오류 | 30분 |
| 비밀번호 10회 오류 | 영구 |
| 90일 미접속 | 자동 |
| 해제 | 지점장+(지점), 본사(지점장) |

## 휴직

휴직 유형에 따라 법적 근거와 급여 처리가 다릅니다.

**육아휴직(법정):** 남녀고용평등법에 따라 만 8세 이하 자녀를 둔 근로자는 최대 1년의 육아휴직을 신청할 수 있으며, 사용자는 이를 거부할 수 없습니다. 육아휴직 급여는 고용보험에서 지급되며, 회사는 별도 급여를 지급하지 않아도 됩니다.

**병가 및 개인 사유 휴직:** 법정 의무는 아니며 회사 정책에 의한 것입니다. 급여는 무급이며, 3개월을 초과하는 장기 휴직은 본사 결재가 필요합니다. 장기 휴직 중 담당 수업과 회원은 지점장이 대체 트레이너에게 재배정합니다.

<Callout type="warning">
**육아휴직**은 법정 권리로, 신청 시 거부할 수 없습니다. 육아휴직을 이유로 불이익한 처우를 하면 남녀고용평등법 위반입니다.
</Callout>

| 사유 | 최대 기간 | 급여 |
|---|---|---|
| 육아 (법정) | 1년 | 고용보험 |
| 병가 | 6개월 | 무급 |
| 개인 사유 | 3개월 | 무급 |

- 담당 재배정: 지점장
- 복직: 7일 전 신청 → 지점장 승인
- 장기 휴직(3개월+): 본사 결재

## CRM에서 직원 관리하기

### 직원 목록 화면

CRM 좌측 메뉴에서 **직원 관리**를 선택하면 전체 직원 목록을 확인할 수 있습니다. 역할(지점장·스태프·FC·트레이너 등)과 상태(재직·수습·휴직·퇴사 등)별 필터로 원하는 직원을 빠르게 찾을 수 있으며, 이름·연락처 키워드로 검색도 가능합니다. 직원 수가 많은 경우 페이지 하단의 페이지네이션을 활용하세요.

### 직원 등록

1. 직원 목록 화면 우측 상단의 **새 직원** 버튼을 클릭하세요.
2. 이름, 역할, 연락처, 입사일, 이메일을 입력하세요.
3. 저장하면 입력한 이메일로 CRM 계정이 자동 생성되고 임시 비밀번호가 발송됩니다.

<Callout type="info">
이메일 주소는 CRM 로그인 아이디로 사용됩니다. 직원 등록 전 이메일이 정확한지 확인하세요.
</Callout>

### 직원 정보 수정

직원 목록에서 해당 직원을 클릭하면 상세 화면으로 이동합니다. 기본 정보(이름·연락처·입사일), 역할, 상태를 수정한 뒤 **저장** 버튼을 클릭하세요.

### 수습 평가

수습 종료 2주 전에 CRM 알림이 발송됩니다. 알림을 확인한 지점장은 해당 직원 상세 화면의 **수습 평가** 탭에서 평가서를 작성하고 본사에 보고하세요. 평가 결과(통과/미통과)는 본사 확인 후 처리됩니다.

### 반기 평가 (6월·12월)

평가 기간이 되면 CRM에서 평가 대상 직원 목록이 표시됩니다. 지점장은 각 직원 상세 화면의 **평가** 탭에서 항목별(수업 품질·회원 관리·매출 기여·근태·팀워크) 점수를 입력하세요. 입력이 완료되면 시스템이 가중치를 적용하여 등급(S/A/B/C/D)을 자동으로 산출합니다.

### 계정 잠금 해제

직원 계정이 잠긴 경우, 지점장 또는 본사는 해당 직원 상세 화면 하단의 **잠금 해제** 버튼을 클릭하여 계정을 해제할 수 있습니다. 해제 전 본인 확인 절차를 반드시 거치세요.

### 휴직 처리

1. 해당 직원 상세 화면에서 **상태** 항목을 클릭하세요.
2. **휴직**을 선택하세요.
3. 휴직 사유와 기간(시작일·예정 복직일)을 입력하고 저장하세요.
4. 3개월을 초과하는 장기 휴직은 본사 결재가 필요합니다.
