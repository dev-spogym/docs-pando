# 근태 / 연차 / 휴가

import { Callout } from 'nextra/components'

근태 관리는 단순한 출결 기록을 넘어, 근로기준법상 사용자의 의무이자 공정한 급여 산정의 기초입니다. 정확한 근태 기록은 초과근무 수당, 연차 산정, 4대보험 신고의 근거가 되므로, 기록의 누락이나 임의 수정은 노동법 위반으로 이어질 수 있습니다. 지점장은 매월 마감 전 근태 데이터를 검토하고 이상 여부를 확인해야 합니다.

## 근태 규정

지각·조퇴·결근의 기준을 명확히 정의함으로써, 직원과 관리자 모두 동일한 기준으로 근태를 평가할 수 있습니다. "근무 시작 + 1분"을 지각 기준으로 설정한 것은 합리적인 여유를 두면서도 습관적 지연을 방지하기 위함입니다.

<Callout type="warning">
근태 기록은 **근로기준법 제48조**에 따라 3년간 보존 의무가 있습니다. 임의 삭제 또는 수정은 법 위반입니다.
</Callout>

| 항목 | 기준 |
|---|---|
| 지각 | 근무 시작 + 1분 이후 |
| 조퇴 | 근무 종료 전 퇴근 (최소 4시간 근무) |
| 결근 | 사전 연락 없이 미출근 |

### 인사 조치

인사 조치는 즉각적인 징계보다 단계적 경고를 통해 직원 스스로 개선할 기회를 주는 원칙을 따릅니다. 구두 경고 → 서면 경고 → 징계위원회 순으로 점진적으로 강화됩니다. 급여 공제는 실제 근무하지 않은 시간에 비례하므로, 근로기준법상 정당한 처리입니다.

| 조건 | 조치 | 급여 |
|---|---|---|
| 지각 월 3회 | 구두 경고 | 없음 |
| 지각 월 5회 | 서면 경고 | 1시간분 공제 |
| 무단 결근 1회 | 서면 경고 | 해당일 공제 |
| 무단 결근 월 3회 | 징계위원회 | 공제 + 상여 감액 |

## 연차 정책

연차 유급휴가는 근로기준법 제60조에 의해 보장되는 법정 권리입니다. 사용자는 요건을 충족한 근로자에게 반드시 연차를 부여해야 하며, 이를 거부하거나 사용을 방해하면 법적 제재를 받습니다.

<Callout type="warning">
**근로기준법 제60조:** 사용자는 1년간 80% 이상 출근한 근로자에게 15일의 유급휴가를 주어야 합니다. 1년 미만 근로자에게는 매월 개근 시 1일의 유급휴가를 부여합니다.
</Callout>

**연차 계산 예시:**
- 입사 첫 해(1년 미만): 매월 1일씩 최대 11일 부여. 예를 들어 1월 입사 후 12월까지 개근하면 11일이 적립됩니다.
- 1년 이상 3년 미만: 15일
- 5년 근속: 15일 + (5년 - 3년) ÷ 2 = 15 + 1 = 16일
- 10년 근속: 15일 + (10년 - 3년) ÷ 2 = 15 + 3 = 18일 (소수점 버림)
- 25일이 최대 한도입니다.

| 근속 | 연차 |
|---|---|
| 1년 미만 | 매월 1일 (최대 11일) |
| 1년+ ~ 3년 | 15일 |
| 3년+ | 15일 + 매 2년마다 1일 (최대 25일) |

### 사용 규칙

연차는 사용일 3일 전에 신청하는 것이 원칙이나, 긴급 상황에서는 당일 신청도 가능합니다. 반차(0.5일)와 반반차(0.25일)를 허용하여 직원이 유연하게 휴가를 활용할 수 있도록 합니다.

| 항목 | 규칙 |
|---|---|
| 신청 | 사용일 3일 전 |
| 승인 | FC/스태프/트레이너 → 지점장, 지점장 → 본사(대표) |
| 반차/반반차 | 허용 |
| 이월 | 미사용 → 1년 이월, 이후 소멸 또는 수당 |

**연차 촉진 제도 (근로기준법 제61조):** 연차 소멸 6개월 전 회사가 서면으로 사용 시기 지정을 촉구했음에도 직원이 사용하지 않으면, 미사용 연차에 대한 수당 지급 의무가 면제됩니다. 반드시 서면 통보 절차를 이행해야 효력이 발생합니다.

### 특별 휴가

특별 휴가는 법정 휴가와 회사 선택 휴가로 구분됩니다. 출산휴가(90일)와 배우자 출산휴가(10일)는 근로기준법 및 남녀고용평등법에 따른 법정 의무로, 거부할 수 없습니다. 결혼·조의 휴가는 회사 정책으로 제공되는 복리후생입니다.

<Callout type="warning">
**출산휴가(90일)**와 **배우자 출산휴가(10일)**는 법정 의무입니다. 직원이 신청하면 반드시 부여해야 하며, 이를 거부하면 근로기준법 위반입니다.
</Callout>

| 종류 | 기간 | 유급 |
|---|---|---|
| 결혼 (본인) | 5일 | 유급 |
| 부모/배우자 사망 | 5일 | 유급 |
| 조부모/형제 사망 | 3일 | 유급 |
| 출산 (본인) | 90일 | 유급 |
| 배우자 출산 | 10일 | 유급 |
| 병가 | 최대 30일 | 무급 |

## CRM에서 근태·연차 관리하기

### 직원 근태 화면

CRM 좌측 메뉴에서 **근태 관리**를 선택하면 날짜별 전 직원의 출근·퇴근·지각·조퇴·결근 현황을 한눈에 확인할 수 있습니다. 날짜 필터를 변경하면 원하는 기간의 근태 데이터를 조회할 수 있습니다.

### 근태 자동 기록

직원이 CRM에 로그인하면 출근이 자동으로 기록되고, 로그아웃하면 퇴근이 자동으로 기록됩니다. 별도로 근태를 입력할 필요 없이 CRM 사용 자체가 출결 기록이 됩니다.

<Callout type="info">
CRM 로그인/로그아웃 시각을 기준으로 지각·조퇴 여부가 자동 판정됩니다. 시스템 오류 등으로 기록이 누락된 경우 지점장에게 수정을 요청하세요.
</Callout>

### 연차 신청

1. CRM 좌측 메뉴에서 **근태 관리**를 선택하세요.
2. **연차 신청** 버튼을 클릭하세요.
3. 신청 기간(시작일·종료일)과 사유를 입력하고 제출하세요.
4. 지점장의 승인이 완료되면 연차가 차감됩니다. 승인 전까지는 **승인 대기** 상태로 표시됩니다.

### 연차 승인 (지점장)

1. CRM 좌측 메뉴에서 **근태 관리 → 연차 요청 목록**을 선택하세요.
2. 각 요청 건에 대해 **승인** 또는 **반려** 버튼을 클릭하세요.
3. 반려 시에는 사유를 입력하면 직원에게 알림이 발송됩니다.

### 잔여 연차 확인

본인 근태 화면 상단에 남은 연차일수가 표시됩니다. 반차·반반차 사용 내역도 소수점 단위로 반영되어 실시간으로 확인할 수 있습니다.

### 특별 휴가 신청

특별 휴가(결혼·조의·출산 등) 신청 시 증빙서류를 첨부해 제출할 수 있습니다. 신청 화면에서 **파일 첨부** 버튼을 클릭하여 관련 서류(청첩장, 사망진단서 등)를 업로드하세요.
