# 근태 / 연차 / 휴가

## 근태 규정

| 항목 | 기준 |
|---|---|
| 지각 | 근무 시작 + 1분 이후 |
| 조퇴 | 근무 종료 전 퇴근 (최소 4시간 근무) |
| 결근 | 사전 연락 없이 미출근 |

### 인사 조치

| 조건 | 조치 | 급여 |
|---|---|---|
| 지각 월 3회 | 구두 경고 | 없음 |
| 지각 월 5회 | 서면 경고 | 1시간분 공제 |
| 무단 결근 1회 | 서면 경고 | 해당일 공제 |
| 무단 결근 월 3회 | 징계위원회 | 공제 + 상여 감액 |

---

## 연차 정책

> 법적 근거: 근로기준법 제60조

| 근속 | 연차 |
|---|---|
| 1년 미만 | 매월 1일 (최대 11일) |
| 1년+ ~ 3년 | 15일 |
| 3년+ | 15일 + 매 2년마다 1일 (최대 25일) |

### 사용 규칙

| 항목 | 규칙 |
|---|---|
| 신청 | 사용일 3일 전 |
| 승인 | FC/스태프/트레이너 → 지점장, 지점장 → 본사(대표) |
| 반차/반반차 | 허용 |
| 이월 | 미사용 → 1년 이월, 이후 소멸 또는 수당 |

### 특별 휴가

| 종류 | 기간 | 유급 |
|---|---|---|
| 결혼 (본인) | 5일 | 유급 |
| 부모/배우자 사망 | 5일 | 유급 |
| 조부모/형제 사망 | 3일 | 유급 |
| 출산 (본인) | 90일 | 유급 |
| 배우자 출산 | 10일 | 유급 |
| 병가 | 최대 30일 | 무급 |
