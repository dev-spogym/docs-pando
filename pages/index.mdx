## 대시보드

로그인 후 첫 화면으로 지점별 핵심 KPI를 한눈에 확인할 수 있습니다.

- **회원 현황 카드**: 총 회원수, 활성, 만료임박(D-7), 만료 회원 수 → 클릭 시 회원 목록으로 이동
- **생일 회원**: 오늘 생일인 회원 최대 5명 표시
- **미납 회원**: 미납액 큰 순서로 최대 5명 표시
- **만료 예정**: D-7 이내 회원 목록
- **당월 매출 차트**: 일별 매출 막대 그래프 (현금/카드/마일리지 구분)
- **매출 요약**: 총 매출, 현금, 카드, 마일리지 금액
- **출석 현황**: 오늘 출석자 수 / 활성 회원 대비 출석률
- **기간 필터**: 오늘/이번주/이번달 선택 가능
- **새로고침**: 수동 새로고침 버튼 + 자동 새로고침(5분) 옵션

## 본사-지점 구조 요약

```
본사 (HQ)
  ├── 회장 ──── 최고 의사결정권자
  ├── 대표 ──── 본사 총괄 + 지점 겸임
  └── 본사 직원 ── 본사 운영/관리
       │
       ├── 지점 A
       │     ├── 지점장
       │     ├── 스태프
       │     ├── FC (피트니스 컨설턴트)
       │     └── 트레이너
       └── 지점 B ...
```

## 핵심 원칙

- **본사 역할**(회장/대표/본사직원)은 전 지점에 대한 CRUD 권한 보유
- **지점 역할**은 소속 지점 데이터만 접근 가능
- 경영/재무/정책에 영향을 미치는 행위는 **본사 결재 승인 필수**
- 급여 확정, 권한 설정, 상품/가격 변경 등은 **본사에서 중앙 관리**
