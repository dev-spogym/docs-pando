# 정원 / 예약 제한 / OT

## 수업 정원 변경

| 항목 | 규칙 |
|---|---|
| 정원 증가 | 지점장 (즉시) |
| 기예약자 이하 축소 | 불가 |
| 기예약자 초과 축소 | 지점장 + 초과 예약자 연락 |

## 예약 제한

| 항목 | 규칙 |
|---|---|
| 동시 예약 | 향후 7일 내 최대 5개 |
| 주간 상한 | 주 7회 |
| 비활성 회원 | 예약 차단 |
| 예약 오픈 | 수업 7일 전 00:00 |

## OT/상담 수업

| 항목 | 규칙 |
|---|---|
| 신규 OT | 권장 (필수 설정 가능) |
| OT 필수 모드 | 미완료 시 PT/GX 예약 차단 |
| OT 담당 | 자동(순번) 또는 수동(지점장) |
| 소요 시간 | 기본 60분 |
| 횟수 차감 | 없음 (무료) |
