# 정원 / 예약 제한 / OT

import { Callout } from 'nextra/components'

수업 운영 규칙은 센터 자원(강사, 시설, 장비)을 최대한 공정하게 배분하고, 안전하고 질 높은 수업 환경을 유지하기 위해 설정됩니다. 예약 제한, 정원 관리, OT 정책은 서로 연결된 시스템으로, 어느 하나가 무너지면 전체 운영 효율이 저하됩니다.

## 수업 정원 변경

정원 변경은 기존에 예약한 회원의 권리를 보호하는 것이 최우선 원칙입니다. 정원을 늘리는 것은 추가 서비스를 제공하는 것이므로 지점장이 자유롭게 결정할 수 있습니다. 반면 정원을 줄이는 것은 이미 예약한 회원에게 불이익을 줄 수 있으므로 엄격히 제한됩니다.

**기예약자 보호 원칙:** 현재 예약 인원이 5명인 수업의 정원을 4명으로 축소하는 것은 허용되지 않습니다. 6명으로 설정된 수업의 정원을 5명으로 줄이려면, 현재 예약자가 5명 이하인 경우에만 가능합니다. 만약 기예약자를 초과하는 축소가 필요한 경우, 지점장은 초과된 예약자에게 개별 연락하여 다른 수업으로 대체 안내해야 합니다.

| 항목 | 규칙 |
|---|---|
| 정원 증가 | 지점장 (즉시) |
| 기예약자 이하 축소 | 불가 |
| 기예약자 초과 축소 | 지점장 + 초과 예약자 연락 |

## 예약 제한

예약 제한의 목적은 특정 회원이 인기 수업을 독점하거나, 취소 없이 다수의 수업을 점유하는 행태를 방지하는 것입니다. 향후 7일 내 최대 5개 수업만 동시 예약 가능하도록 제한함으로써, 대기자가 있는 수업에서 자리를 공정하게 배분할 수 있습니다. 주간 상한(7회)은 과도한 운동으로 인한 부상 위험도 간접적으로 예방하는 효과가 있습니다.

**예약 오픈 정책의 의미:** 수업 7일 전 00:00에 예약이 오픈됩니다. 특정 회원이 수업 정보를 미리 알고 독점 예약하는 것을 방지하고, 모든 활성 회원에게 동등한 예약 기회를 제공하기 위한 장치입니다.

| 항목 | 규칙 |
|---|---|
| 동시 예약 | 향후 7일 내 최대 5개 |
| 주간 상한 | 주 7회 |
| 비활성 회원 | 예약 차단 |
| 예약 오픈 | 수업 7일 전 00:00 |

## OT/상담 수업

OT(오리엔테이션)는 신규 회원이 센터 시설, 장비 사용법, 기본 운동 자세를 익히는 필수적인 첫 단계입니다. OT 없이 바로 PT나 GX에 참여하면 잘못된 자세나 장비 오사용으로 인한 부상 위험이 높아집니다. 특히 웨이트 트레이닝 장비는 초보자에게 척추·관절 부상의 원인이 될 수 있으므로, OT 필수 모드를 활성화하면 OT 미완료 회원의 PT/GX 예약 자체가 차단됩니다.

<Callout type="info">
OT 필수 모드는 센터 설정에서 활성화/비활성화할 수 있습니다. 활성화 시 신규 회원은 OT를 완료해야만 일반 수업 예약이 가능합니다. OT는 횟수 차감 없이 무료로 제공됩니다.
</Callout>

**OT 담당 배정 방식:**
- 자동(순번): 시스템이 트레이너 순번을 자동으로 배정하여 업무를 균등하게 분배합니다.
- 수동(지점장): 특정 트레이너의 전문 분야나 회원의 요청을 고려해 지점장이 직접 배정합니다.

| 항목 | 규칙 |
|---|---|
| 신규 OT | 권장 (필수 설정 가능) |
| OT 필수 모드 | 미완료 시 PT/GX 예약 차단 |
| OT 담당 | 자동(순번) 또는 수동(지점장) |
| 소요 시간 | 기본 60분 |
| 횟수 차감 | 없음 (무료) |

## CRM에서 정원/시간표/OT 관리하기

### 정원 변경

수업 상세 화면에서 정원을 수정할 수 있습니다. 현재 예약자 수 이하로 줄이는 것은 불가하며, 초과 예약자가 있을 경우 지점장이 해당 회원에게 개별 연락한 후 변경해야 합니다.

### 시간표 관리

설정 → 시간표 메뉴에서 요일별 수업 배치를 관리할 수 있습니다. 시간표에서 수업을 드래그하여 시간대를 변경하거나, 새 수업 슬롯을 추가할 수 있습니다.

### OT 배정

신규 회원 등록 시 시스템이 트레이너 순번에 따라 OT 담당자를 자동 추천합니다. 지점장은 회원 상세 → OT 탭에서 담당 트레이너를 수동으로 변경할 수 있습니다.

### 예약 현황 대시보드

대시보드 → 수업 현황에서 수업별 예약률과 인기 시간대 통계를 확인할 수 있습니다. 예약률이 낮은 수업이나 대기자가 많은 시간대를 파악해 시간표 조정에 활용하세요.
