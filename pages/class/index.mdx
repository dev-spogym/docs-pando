# 예약 / 취소 정책

## 예약 취소 규칙

| 취소 시점 | 페널티 |
|---|---|
| 수업 24시간 전까지 | 없음 |
| 수업 12~24시간 전 | 경고 1회 |
| 수업 4~12시간 전 | 횟수 1회 차감 (횟수제) |
| 수업 4시간 이내 | 노쇼와 동일 |

### 취소 권한

| 행위 | 권한 |
|---|---|
| 본인 예약 취소 | 회원(앱) 또는 스태프+ |
| 타인 예약 취소 | 지점장+ |
| 전체 수업 취소 | 지점장+ (보강 의무 발생) |

### 월간 취소 제한

| 조건 | 제한 |
|---|---|
| 월 3회까지 | 자유 취소 |
| 월 4~5회 | 경고 알림 |
| 월 6회+ | 추가 예약 제한 (지점장 해제) |
