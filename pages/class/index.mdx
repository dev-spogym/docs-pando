# 예약 / 취소 정책

import { Callout } from 'nextra/components'

예약 취소 정책은 노쇼 방지와 회원 편의 사이의 균형을 맞추기 위해 설계되었습니다. 수업 예약은 다른 회원의 참여 기회를 제한하는 행위이므로, 취소가 늦을수록 강사와 센터 운영에 미치는 영향이 커집니다. 반면 지나치게 엄격한 정책은 회원의 불만을 야기할 수 있습니다. 따라서 취소 시점에 따라 단계적으로 페널티를 부과하여, 합리적인 사전 취소를 유도합니다.

## 예약 취소 규칙

시간대별 페널티 차등의 이유는 다음과 같습니다. 수업 24시간 전 취소는 대기자를 통한 재배정이 충분히 가능하여 운영 손실이 거의 없습니다. 12~24시간 전 취소는 재배정 가능성이 낮아지므로 경고로 주의를 환기합니다. 4~12시간 전 취소는 이미 강사와 시설 준비가 시작된 시점이므로 횟수 차감으로 실질적 부담을 줍니다. 4시간 이내 취소는 사실상 노쇼와 동일한 영향을 미치므로 동일하게 처리합니다.

| 취소 시점 | 페널티 |
|---|---|
| 수업 24시간 전까지 | 없음 |
| 수업 12~24시간 전 | 경고 1회 |
| 수업 4~12시간 전 | 횟수 1회 차감 (횟수제) |
| 수업 4시간 이내 | 노쇼와 동일 |

### 취소 권한

| 행위 | 권한 |
|---|---|
| 본인 예약 취소 | 회원(앱) 또는 스태프+ |
| 타인 예약 취소 | 지점장+ |
| 전체 수업 취소 | 지점장+ (보강 의무 발생) |

### 월간 취소 제한

월간 취소 제한은 특정 회원이 반복적으로 예약 후 취소하는 행동 패턴을 방지하기 위한 운영 장치입니다. 예약이 가득 찬 인기 수업에서 무분별한 취소는 다른 회원의 수강 기회를 박탈합니다. 월 3회까지는 자유롭게 취소할 수 있으며, 4~5회부터 경고 알림을 발송하여 스스로 인지하도록 유도합니다. 6회 이상이 되면 추가 예약이 제한되며, 지점장이 상황을 확인한 후 해제 여부를 결정합니다.

<Callout type="info">
월간 취소 횟수는 매월 1일에 초기화됩니다. 제한 해제는 지점장만 가능하며, 불가피한 사유가 있는 경우 회원이 지점에 직접 방문하여 설명하면 지점장 판단으로 해제할 수 있습니다.
</Callout>

| 조건 | 제한 |
|---|---|
| 월 3회까지 | 자유 취소 |
| 월 4~5회 | 경고 알림 |
| 월 6회+ | 추가 예약 제한 (지점장 해제) |

## CRM에서 수업 관리하기

### 캘린더 화면

수업 캘린더는 일/주/월 뷰를 지원하며, 수업 유형에 따라 색상으로 구분됩니다.

| 수업 유형 | 색상 |
|---|---|
| PT (개인 트레이닝) | 파랑 |
| GX (그룹 운동) | 초록 |
| OT (오리엔테이션) | 노랑 |

### 수업 등록

캘린더에서 원하는 시간 슬롯을 클릭하면 수업 등록 화면이 열립니다. 수업 유형, 담당 강사, 정원을 입력하고 저장하면 캘린더에 즉시 반영됩니다.

**수업 템플릿 활용:** 매주 반복되는 수업 패턴은 템플릿으로 저장해 두면 일괄 등록이 가능합니다. 설정 → 수업 템플릿에서 관리할 수 있습니다.

### 예약 관리

회원이 앱에서 수업을 예약하면 CRM 캘린더에 실시간으로 반영됩니다. 캘린더에서 수업을 클릭하면 현재 예약자 목록과 대기자 현황을 확인할 수 있습니다.

### 출석 처리

출석은 두 가지 방법으로 처리할 수 있습니다.

- **키오스크 자동 체크인:** 회원이 키오스크에서 인증하면 출석이 자동으로 기록됩니다.
- **CRM 수동 체크인:** 수업 상세 화면 → 예약자 목록에서 해당 회원의 출석 상태를 직접 변경합니다.

### 출석 현황 조회

매출 → 출석 현황에서 날짜별 또는 회원별로 출석 이력을 필터링해 조회할 수 있습니다. 출석률 통계도 함께 제공되므로 수업 운영 현황 파악에 활용하세요.
