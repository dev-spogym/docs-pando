# 노쇼 / 대기자 / 보강

import { Callout } from 'nextra/components'

노쇼(No-show)는 예약을 해놓고 아무 연락 없이 수업에 불참하는 행위입니다. 노쇼가 반복되면 강사의 수업 준비가 낭비되고, 대기 중인 다른 회원의 기회가 박탈됩니다. 페널티 정책은 즉각적인 제재보다 점진적 경고와 제한을 통해 회원 스스로 행동을 개선하도록 유도하는 데 목적이 있습니다.

## 노쇼 페널티

페널티 단계별 설계 원칙은 다음과 같습니다. 1단계(SMS 경고)는 단순 실수일 수 있으므로 인지를 촉구합니다. 2단계는 담당 코치에게도 알려 개인 상담 기회를 만듭니다. 3단계부터는 예약 제한과 횟수 차감이 함께 적용되어 실질적 불이익이 발생합니다. 4단계는 제한 기간이 2배로 늘어나 반복 노쇼를 강하게 억제합니다. 이 구조는 "경고 → 제한 → 차감"으로 점진적으로 강화되어, 과도한 제재로 인한 회원 이탈을 최소화하면서도 노쇼를 실질적으로 줄이는 효과를 냅니다.

| 단계 | 조건 | 페널티 |
|---|---|---|
| 1단계 | 월 1회 | SMS 경고 |
| 2단계 | 월 2회 | SMS + 담당 코치 알림 |
| 3단계 | 월 3회 | 7일 예약 제한 + 횟수 1회 차감 |
| 4단계 | 월 4회+ | 14일 예약 제한 + 추가 차감 |

### 면제 사유

면제 사유 처리 시 주의사항: 면제는 회원이 불참을 사전에 알리지 못한 불가항력적 상황에 한정됩니다. 단순 깜빡임이나 개인 일정 변경은 면제 대상이 아닙니다. 지점장은 증빙 서류를 확인한 후 시스템에 면제 사유를 입력하고 AuditLog에 기록해야 합니다. "센터 공지 부재"는 시스템이 자동으로 확인하여 처리하므로 지점장의 별도 조치가 필요 없습니다.

<Callout type="warning">
면제 처리는 지점장 전용입니다. 스태프나 FC가 구두로 "면제해 줬다"고 해도 시스템에 반영되지 않으면 페널티는 유지됩니다.
</Callout>

| 사유 | 증빙 | 면제 권한 |
|---|---|---|
| 긴급 의료 | 진료 확인서 | 지점장 |
| 교통 사고/천재지변 | 관련 증빙 | 지점장 |
| 센터 공지 부재 | 시스템 확인 | 자동 |

## 대기자(Waitlist) 정책

대기자 시스템은 정원이 가득 찬 인기 수업에서 취소 발생 시 자동으로 다음 회원에게 자리를 배정하는 기능입니다. 이를 통해 수업 정원의 실질 가동률을 높이고, 강사 입장에서도 빈자리 없이 수업을 진행할 수 있어 인센티브 극대화에 기여합니다. 수락 기한(2시간)을 두는 이유는 알림을 받지 못한 회원이 무기한 자리를 점유하는 것을 방지하기 위함입니다. 대기 만료를 수업 12시간 전으로 설정한 것은, 이 시점 이후의 배정은 회원이 준비할 시간이 부족할 수 있기 때문입니다.

| 항목 | 규칙 |
|---|---|
| 대기 등록 | 정원 초과 시 옵션 제공 |
| 순서 | 선착순 |
| 자동 배정 | 취소 시 1순위에게 배정 |
| 수락 기한 | 2시간 (미수락 → 다음) |
| 대기 만료 | 수업 12시간 전 |
| 최대 인원 | 정원의 50% |

## 보강 수업

보강 수업 정책의 핵심은 **귀책 주체에 따른 공정한 처리**입니다. 강사 또는 센터 사정으로 수업이 취소된 경우(센터 귀책)에는 회원이 이미 비용을 지불했으므로 보강 1회가 의무로 발생합니다. 반면 회원이 취소하거나 노쇼한 경우(회원 귀책)에는 보강 의무가 발생하지 않습니다. 보강을 14일 내에 이행하지 못하면 회원의 횟수를 1회 복원하여, 보강 불이행으로 인한 회원 불이익을 방지합니다.

<Callout type="info">
보강 수업은 원래 수업과 동일한 강사가 진행하는 것이 원칙입니다. 불가피하게 다른 강사가 진행할 경우 회원에게 사전 안내해야 합니다.
</Callout>

| 항목 | 규칙 |
|---|---|
| 보강 의무 | 강사/센터 사정 취소 시 1회 |
| 유효 기간 | 취소일로부터 14일 |
| 미이행 | 유효 기간 경과 → 횟수 1회 복원 |
| 등록 | 트레이너(본인) 또는 지점장 |

## CRM에서 노쇼/대기자/보강 관리하기

### 노쇼 처리

수업이 종료되면 미출석 회원은 자동으로 노쇼 처리됩니다. 필요한 경우 수업 상세 화면 → 예약자 목록에서 수동으로 노쇼를 등록할 수도 있습니다.

**노쇼 이력 조회:** 회원 상세 → 예약 탭에서 해당 회원의 노쇼 이력과 누적 페널티 단계를 확인할 수 있습니다.

### 대기자 관리

수업 상세 화면의 대기자 탭에서 현재 대기 중인 회원 목록과 순서를 확인할 수 있습니다. 예약 취소가 발생하면 대기 1순위 회원에게 자동으로 알림이 발송되며, 수락 여부가 CRM에 실시간으로 반영됩니다.

### 보강 등록

강사 또는 센터 사정으로 수업이 취소되면 보강 수업이 자동으로 생성됩니다. 캘린더 → 해당 수업 → "보강 등록" 버튼을 눌러 보강 일정을 직접 지정할 수도 있습니다. 보강 수업은 원래 수업과 동일한 강사로 자동 배정되며, 불가피한 경우 강사 변경 후 회원에게 사전 안내해야 합니다.
