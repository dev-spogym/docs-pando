# 노쇼 / 대기자 / 보강

## 노쇼 페널티

| 단계 | 조건 | 페널티 |
|---|---|---|
| 1단계 | 월 1회 | SMS 경고 |
| 2단계 | 월 2회 | SMS + 담당 코치 알림 |
| 3단계 | 월 3회 | 7일 예약 제한 + 횟수 1회 차감 |
| 4단계 | 월 4회+ | 14일 예약 제한 + 추가 차감 |

### 면제 사유

| 사유 | 증빙 | 면제 권한 |
|---|---|---|
| 긴급 의료 | 진료 확인서 | 지점장 |
| 교통 사고/천재지변 | 관련 증빙 | 지점장 |
| 센터 공지 부재 | 시스템 확인 | 자동 |

---

## 대기자(Waitlist) 정책

| 항목 | 규칙 |
|---|---|
| 대기 등록 | 정원 초과 시 옵션 제공 |
| 순서 | 선착순 |
| 자동 배정 | 취소 시 1순위에게 배정 |
| 수락 기한 | 2시간 (미수락 → 다음) |
| 대기 만료 | 수업 12시간 전 |
| 최대 인원 | 정원의 50% |

---

## 보강 수업

| 항목 | 규칙 |
|---|---|
| 보강 의무 | 강사/센터 사정 취소 시 1회 |
| 유효 기간 | 취소일로부터 14일 |
| 미이행 | 유효 기간 경과 → 횟수 1회 복원 |
| 등록 | 트레이너(본인) 또는 지점장 |
