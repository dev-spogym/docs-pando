# 휴회(HOLDING) 정책

import { Callout } from 'nextra/components'

휴회 정책은 회원 유지와 이탈 방지를 위한 핵심 장치입니다. 부상, 여행, 업무 과중 등 불가피한 사정으로 잠시 이용이 어려운 회원이 계약을 해지하는 대신 이용을 일시 중단할 수 있도록 합니다. 회원 입장에서는 이미 납부한 금액을 보호받으면서 나중에 다시 이용할 수 있고, 센터 입장에서는 회원을 유지하여 장기적인 관계를 이어갈 수 있습니다. 휴회 기간만큼 이용 기간이 자동 연장되므로 회원은 실제 이용한 만큼만 비용을 지불하게 됩니다.

## 기본 규칙

휴회 신청을 받기 위한 조건은 회원이 실제로 이용 가능한 상태인지를 검증합니다. 미수금이 있는 상태에서 휴회를 허용하면 미납금을 방치한 채 이용을 중단하는 악용 사례가 생길 수 있습니다. 잔여 기간 30일 이상 조건은 이미 거의 소진된 이용권에 대한 무의미한 휴회를 방지합니다.

| 항목 | 규칙 |
|---|---|
| 신청 조건 | ACTIVE 상태, 미수금 없음, 잔여 기간 30일+ |
| 최소/최대 기간 | 7일 ~ 90일 |
| 연간 횟수 | 최대 2회 |
| 이용 기간 | 휴회 일수만큼 자동 연장 |
| 시설 이용 | 전면 차단 (출입/예약 불가) |
| 자동 복귀 | 종료일 00:00 자동 ACTIVE 전환 |
| 조기 복귀 | 가능 (잔여 휴회 기간 소멸) |

<Callout type="warning">
조기 복귀 시 **잔여 휴회 기간은 소멸**됩니다. 예를 들어 30일 휴회를 신청했지만 10일 만에 조기 복귀하면, 남은 20일은 이용 기간에 추가되지 않습니다. 복귀 전에 회원에게 반드시 안내해야 합니다.
</Callout>

| 행위 | 권한 |

## 특수 휴회 (90일 초과)

일반 휴회의 최대 기간은 90일이지만, 의료적 사유나 군 입대처럼 장기 이용 불가 상황에는 90일을 초과하는 특수 휴회가 허용됩니다. 특수 휴회는 반드시 서류를 제출하고 지점장의 승인을 받아야 합니다.

**각 사유별 실무 처리 절차:**

- **의료(수술/입원)**: 병원 진단서 또는 입원 확인서를 접수합니다. 수술 후 회복 기간을 고려하여 지점장이 적절한 기간을 협의합니다. 최대 180일까지 인정됩니다.
- **군 입대**: 입영통지서를 확인하고 복무 기간 전체를 휴회로 처리합니다. 전역 후 복귀 시 자동 ACTIVE 전환되며, 필요시 조기 복귀 신청도 가능합니다.
- **임신/출산**: 임신확인서 또는 출생증명서를 접수합니다. 출산 후 회복을 포함하여 최대 365일까지 허용됩니다. 이 경우 연간 횟수 제한(2회)이 적용되지 않습니다.
- **해외 장기 체류**: 항공권과 비자 사본으로 장기 체류를 확인합니다. 출국일부터 귀국 예정일까지 최대 180일까지 인정됩니다.

| 사유 | 최대 기간 | 필요 서류 | 승인 |
|---|---|---|---|
| 의료 (수술/입원) | 180일 | 진단서 | 지점장 |
| 군 입대 | 복무 기간 | 입영통지서 | 지점장 |
| 임신/출산 | 365일 | 임신확인서/출생증명서 | 지점장 |
| 해외 장기 체류 | 180일 | 항공권/비자 | 지점장 |

## CRM에서 휴회 처리하기

### 휴회 신청 접수

1. 회원 상세 화면 우측 상단의 **상태 변경** 버튼을 클릭합니다.
2. 드롭다운에서 **휴회**를 선택합니다.
3. **시작일**과 **종료일**을 입력합니다(최소 7일, 최대 90일).
4. **사유**를 선택합니다(일반 사유 / 의료 / 군 입대 / 임신·출산 / 해외 체류).
5. 특수 휴회(90일 초과)의 경우 **증빙서류 첨부** 버튼으로 파일을 업로드합니다.
6. **저장**을 누르면 상태가 HOLDING으로 전환되고 이용 기간이 자동 연장됩니다.

<Callout type="info">
특수 휴회는 저장 후 지점장 승인 대기 상태가 됩니다. 지점장이 승인해야 최종 처리됩니다.
</Callout>

### 조기 복귀 처리

휴회 중인 회원이 종료일 이전에 복귀하려면 아래와 같이 처리하세요.

1. 회원 상세 화면에서 현재 상태(HOLDING) 옆의 **조기 복귀** 버튼을 클릭합니다.
2. 복귀 확인 팝업에서 **확인**을 누르면 즉시 ACTIVE로 전환됩니다.

<Callout type="warning">
조기 복귀 처리 전에 반드시 회원에게 **잔여 휴회 기간이 소멸**된다는 사실을 안내하세요.
</Callout>

## 자동 복귀 프로세스

<Callout type="info">
자동 복귀는 종료일 00:00에 배치 작업으로 처리됩니다. 복귀 3일 전에 SMS로 사전 안내가 발송되므로 회원이 미리 준비할 수 있습니다.
</Callout>

```
신청 접수 → 조건 검증(시스템) → 상태 변경(HOLDING) → 이용 기간 연장
→ 종료 3일 전 알림(SMS) → 종료일 자동 복귀(ACTIVE) → 복귀 알림
```

자동 복귀가 실패하는 경우(예: 미수금 발생, 정지 사유 발생)에는 시스템이 스태프에게 알림을 보내 수동 처리가 필요함을 안내합니다.
