# 양도 / 이관 정책

## 양도와 이관의 차이

**양도**는 회원이 본인의 이용권을 **다른 사람**에게 넘기는 것입니다. 예를 들어 3개월 이용권을 구매했지만 사정상 이용이 어려워진 회원이 지인에게 잔여 이용권을 넘기는 경우입니다. 양도는 같은 지점 내에서만 가능하며, 1회에 한해 허용됩니다.

**이관**은 회원이 **다른 지점**으로 소속을 옮기는 것입니다. 이사, 직장 이전 등으로 기존 지점 이용이 어려워졌을 때 잔여 이용권을 유지하면서 가까운 지점으로 옮길 수 있습니다.

## 회원 양도

### 기본 규칙

양도 수수료는 회원권을 되팔거나 타인에게 이전하는 행위에 대한 관리 비용입니다. 잔여 가치 기준으로 산정하여 실질적인 가치에 비례하도록 설계되어 있습니다.

**양도 수수료 산정 예시:**
- 이용권 구매 금액: 100만 원
- 잔여 기간: 전체의 60% (예: 3개월 중 1.8개월 남음)
- 잔여 가치: 100만 원 × 60% = 60만 원
- 수수료: 60만 원 × 10% = **6만 원**
- 수수료 범위: 최소 3만 원 ~ 최대 10만 원 적용

<Callout type="info">
위 예시에서 잔여 가치가 20만 원이라면 수수료는 2만 원이 되지만, 최소 수수료 3만 원이 적용됩니다. 잔여 가치가 200만 원이라면 수수료는 20만 원이 되지만, 최대 10만 원이 적용됩니다.
</Callout>

| 항목 | 규칙 |
|---|---|
| 양도 조건 | ACTIVE, 잔여 30일+/5회+, 미수금 없음 |
| 양도 대상 | 동일 지점 내 |
| 수수료 | 잔여 가치의 10% (최소 3만, 최대 10만) |
| 양도 횟수 | 1회 (재양도 불가) |
| 신청 방법 | 양도인+양수인 방문 필수 |
| 승인 | 지점장 |

### 이전/비이전 항목

마일리지·쿠폰은 원 회원이 특정 거래 관계에서 얻은 혜택이므로 양수인에게 이전되지 않습니다. 수업 이력과 체성분 기록은 개인의 건강 정보이므로 역시 이전하지 않습니다.

| 이전 | 비이전 |
|---|---|
| 잔여 기간, 잔여 횟수, 락커(선택) | 마일리지, 쿠폰, 수업 이력, 체성분, 담당 코치 |

### 프로세스

양도인과 양수인이 반드시 함께 방문해야 하는 이유는 본인 의사 확인과 신분 확인을 위해서입니다. 대리 신청이나 전화·온라인 신청은 허용하지 않습니다.

```
양도인+양수인 방문 → 신분 확인 → 신청서 → 수수료 결제
→ 지점장 승인 → 이용권 이전 → AuditLog
```

### CRM에서 양도 처리하기

1. 양도인 회원 상세 화면 우측 상단의 **양도** 버튼을 클릭합니다.
2. 양수인 정보(이름, 연락처)를 입력합니다. 기존 회원이면 검색으로 찾을 수 있습니다.
3. 자동 산정된 **양도 수수료**를 확인하고 POS에서 결제를 진행합니다.
4. **지점장 승인** 후 이용권이 양수인에게 이전되고 양도인은 WITHDRAWN 상태가 됩니다.

<Callout type="info">
양도 수수료는 잔여 가치의 10%(최소 3만 원, 최대 10만 원)로 시스템이 자동 계산합니다. 처리 전 화면에서 금액을 반드시 확인하세요.
</Callout>

## 회원 이관

### 이관 유형 3종과 사용 시나리오

이관 유형은 회원의 상황과 수신 지점의 상품 구성에 따라 선택합니다.

**KEEP_TICKET (잔여 유지)**
가장 일반적인 이관 방식입니다. 잔여 기간과 횟수를 그대로 유지한 채 수신 지점으로 옮깁니다. 두 지점의 동일 상품 가격이 다를 경우 차액을 정산합니다.
- 사용 상황: 이사로 인해 강남점에서 판교점으로 옮기는 경우. 잔여 2개월이 그대로 판교점에서 유지됩니다.

**REFUND_NEW (환불 후 신규)**
원 지점에서 환불을 받고 수신 지점에서 새로 등록하는 방식입니다. 두 지점의 상품 구성이 완전히 달라 잔여 이용권을 직접 이전하기 어려울 때 사용합니다.
- 사용 상황: 강남점의 6개월 헬스 이용권이 역삼점에는 없는 경우. 강남점에서 잔여 금액을 환불받고 역삼점의 상품으로 새로 계약합니다.

**EXHAUST_THEN (소진 후 이관)**
현재 이용권을 다 쓴 후 수신 지점에서 새로 등록하는 방식입니다. 이사가 예정되어 있지만 아직 잔여 이용 기간이 많이 남아 기존 지점을 당분간 계속 이용하려는 경우에 적합합니다.
- 사용 상황: 두 달 뒤 이사 예정인 회원이 강남점 잔여 이용권을 모두 소진한 후 판교점으로 신규 등록합니다.

### 이관 유형 요약

| 유형 | 설명 |
|---|---|
| **KEEP_TICKET** | 잔여 기간/횟수 그대로 이관. 가격 차이 시 차액 결제 |
| **REFUND_NEW** | 환불 후 수신 지점에서 신규 등록 |
| **EXHAUST_THEN** | 기존 이용권 소진 후 수신 지점 등록 |

### CRM에서 이관 처리하기

1. 회원 상세 화면 우측 상단의 **이관** 버튼을 클릭합니다.
2. 이관 유형을 선택합니다(KEEP_TICKET / REFUND_NEW / EXHAUST_THEN).
3. **수신 지점**을 검색하여 선택합니다.
4. 서면 동의서 확인 후 **신청 완료**를 누릅니다.
5. 수신 지점 지점장의 승인을 대기합니다. 승인 완료 시 자동으로 이관이 처리됩니다.

<Callout type="info">
이관 신청 후 수신 지점 지점장이 승인하기 전까지는 회원 상태가 변경되지 않습니다. 승인 현황은 회원 상세 화면의 **이관 이력** 탭에서 확인할 수 있습니다.
</Callout>

### 공통 규칙

<Callout type="warning">
이관은 반드시 **회원의 서면 동의**와 **수신 지점 지점장의 승인**이 필요합니다. 수신 지점이 PAUSED 또는 INACTIVE 상태인 경우 이관이 차단됩니다.
</Callout>

| 항목 | 규칙 |
|---|---|
| 회원 동의 | 서면 동의 필수 |
| 수신 지점 승인 | 수신 지점 지점장 승인 |
| 이관 불가 조건 | 미수금, 블랙리스트, 수신 지점 비활성 |
| 기존 데이터 | 원 지점 보존 (사본 연계) |
| 이관 권한 | 지점장+ |
