# 양도 / 이관 정책

## 회원 양도

### 기본 규칙

| 항목 | 규칙 |
|---|---|
| 양도 조건 | ACTIVE, 잔여 30일+/5회+, 미수금 없음 |
| 양도 대상 | 동일 지점 내 |
| 수수료 | 잔여 가치의 10% (최소 3만, 최대 10만) |
| 양도 횟수 | 1회 (재양도 불가) |
| 신청 방법 | 양도인+양수인 방문 필수 |
| 승인 | 지점장 |

### 이전/비이전 항목

| 이전 | 비이전 |
|---|---|
| 잔여 기간, 잔여 횟수, 락커(선택) | 마일리지, 쿠폰, 수업 이력, 체성분, 담당 코치 |

### 프로세스

```
양도인+양수인 방문 → 신분 확인 → 신청서 → 수수료 결제
→ 지점장 승인 → 이용권 이전 → AuditLog
```

## 회원 이관

### 이관 유형 (3종)

| 유형 | 설명 |
|---|---|
| **KEEP_TICKET** | 잔여 기간/횟수 그대로 이관. 가격 차이 시 차액 결제 |
| **REFUND_NEW** | 환불 후 수신 지점에서 신규 등록 |
| **EXHAUST_THEN** | 기존 이용권 소진 후 수신 지점 등록 |

### 공통 규칙

| 항목 | 규칙 |
|---|---|
| 회원 동의 | 서면 동의 필수 |
| 수신 지점 승인 | 수신 지점 지점장 승인 |
| 이관 불가 조건 | 미수금, 블랙리스트, 수신 지점 비활성 |
| 기존 데이터 | 원 지점 보존 (사본 연계) |
| 이관 권한 | 지점장+ |
