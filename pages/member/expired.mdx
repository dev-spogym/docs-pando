# 만료 / 휴면 정책

## 만료(EXPIRED) 자동 전환

### 만료 조건

| 상품 유형 | 조건 | 처리 시점 |
|---|---|---|
| 기간제 | endDate 도래 | 매일 00:00 배치 |
| 횟수제 | 잔여 횟수 0 | 즉시 (수업 완료 시) |
| 혼합형 | 기간/횟수 중 먼저 도래 | 상동 |

### 사전 알림

| 시점 | 채널 |
|---|---|
| 30일 전 | 앱 푸시 |
| 14일 전 | SMS + 앱 푸시 |
| 7일 전 | SMS |
| 3일 전 | SMS |
| 당일 | SMS |

### 만료 후 유예

| 항목 | 규칙 |
|---|---|
| 유예 기간 | 만료 후 7일 |
| 유예 중 출입 | 차단 |
| 유예 중 재등록 | 기존 데이터 연계, 등록비 면제 |
| 유예 후 재등록 | 신규 등록 절차, 등록비 발생 |
| 데이터 | 수업 이력/체성분/상담 기록 영구 보존 |

---

## 휴면(DORMANT) 전환

| 항목 | 규칙 |
|---|---|
| 전환 조건 | EXPIRED 상태에서 90일 경과 |
| 처리 | 매일 배치 자동 전환 |
| 전환 전 알림 | 60일, 75일, 85일 경과 시 SMS |
| 데이터 파기 | 휴면 후 3년 경과 시 (개인정보보호법) |
| 파기 전 알림 | 파기 30일 전 SMS |
| 휴면 해제 | 재등록 시 자동 ACTIVE |
