# 정지(SUSPENDED) 정책

import { Callout } from 'nextra/components'

정지는 회원이 의무(납부, 규정 준수)를 이행하지 않거나 다른 회원에게 피해를 주는 행위를 했을 때 시설 이용을 차단하는 조치입니다. 정지 사유에 따라 이용 기간 처리 방식이 다르며, 해제 조건과 절차도 구분됩니다.

## 정지 사유 분류

| 분류 | 사유 | 기간 | 부과 |
|---|---|---|---|
| 미납 | 미수금 30일 초과 | 수납 시까지 | 시스템 자동 |
| 자동결제 실패 | 3회 연속 실패 | 결제 성공 시까지 | 시스템 자동 |
| 경미 위반 | 시설 규칙 위반 | 7일 | 지점장 |
| 중대 위반 | 시설 파손, 위협 | 14~30일 | 지점장 |
| 폭력/성범죄 | 폭행 등 | 영구 (→ 블랙리스트) | 지점장 + 본사 |

## 미납 정지와 규정 위반 정지의 이용 기간 처리 차이

<Callout type="info">
정지 사유에 따라 이용 기간이 다르게 처리됩니다. 이 차이는 회원의 귀책 여부에 따른 합리적 설계입니다.
</Callout>

**미납 정지**: 이용 기간이 **일시 정지**됩니다. 회원이 납부하지 않아 발생한 정지이므로, 정지 기간 동안 이용권의 남은 기간이 소진되지 않습니다. 미납금을 납부하면 정지가 해제되고 잔여 이용 기간이 그대로 유지됩니다. 예를 들어 잔여 60일이 남은 상태에서 30일간 미납 정지를 받았다면, 해제 후에도 잔여 60일이 그대로 남아 있습니다.

**규정 위반 정지**: 이용 기간이 **계속 소진**됩니다. 회원 본인의 귀책(규칙 위반)으로 발생한 정지이므로, 정지 기간 동안도 이용권의 기간이 정상적으로 흐릅니다. 이는 위반 행위에 대한 실질적인 패널티 역할을 합니다.

| 항목 | 규칙 |
|---|---|
| 출입 | 차단 |
| 예약 | 불가 |
| 결제 | 미납 해소 목적만 |
| 이용 기간 (미납) | 기간 정지 (복귀 시 잔여 유지) |
| 이용 기간 (위반) | 계속 소진 |
| 알림 | 즉시 SMS (사유 + 해제 조건) |

## 자동 정지 → 자동 해제 프로세스

미납과 자동결제 실패는 시스템이 자동으로 처리합니다. 사람의 개입 없이 조건이 충족되면 즉시 상태가 변경됩니다.

**미납 자동 정지 흐름:**
```
미수금 발생 → 10일 경과: 1차 납부 독촉 SMS
→ 20일 경과: 2차 독촉 + 정지 예고 SMS
→ 30일 경과: 자동 SUSPENDED 전환 + 정지 통보 SMS
→ 회원 미납금 납부 → 시스템 즉시 ACTIVE 복귀 + 해제 SMS
```

**자동결제 실패 흐름:**
```
자동결제 실패 → 1~2회: 재시도 + 결제 수단 변경 요청 SMS
→ 3회 연속 실패: 자동 SUSPENDED 전환
→ 결제 성공: 시스템 즉시 ACTIVE 복귀
```

## CRM에서 정지 처리하기

### 미납 자동 정지

미납으로 인한 정지는 시스템이 자동으로 처리하므로 별도 수동 조작이 필요 없습니다. 정지 전 독촉 SMS가 자동 발송되며, 회원이 미납금을 납부하면 즉시 ACTIVE로 복귀됩니다.

정지 이력은 회원 상세 화면의 **상태 이력** 탭에서 확인할 수 있습니다.

### 수동 정지 처리

규정 위반 등 수동 정지가 필요한 경우 아래와 같이 처리하세요.

1. 회원 상세 화면 우측 상단의 **상태 변경** 버튼을 클릭합니다.
2. 드롭다운에서 **정지**를 선택합니다.
3. **사유**(경미 위반 / 중대 위반 / 폭력·성범죄)와 **기간**을 입력합니다.
4. **저장**을 누르면 즉시 SUSPENDED 상태로 전환되고 회원에게 SMS가 발송됩니다.

<Callout type="info">
정지 처리 권한은 지점장 이상입니다. 스태프는 정지 처리를 직접 할 수 없으며, 지점장에게 요청해야 합니다.
</Callout>

### 수동 정지 해제

미납 해제(시스템 자동) 외에 중대 위반 정지는 지점장이 직접 해제해야 합니다.

1. 회원 상세 화면에서 현재 상태(SUSPENDED) 옆의 **정지 해제** 버튼을 클릭합니다.
2. 해제 사유를 입력하고 **확인**을 누르면 ACTIVE로 전환됩니다.

### 정지 이력 조회

회원 상세 화면 → **상태 이력** 탭에서 과거 정지 기록(사유, 기간, 처리자)을 확인할 수 있습니다.

## 해제 규칙

중대 위반은 기간이 만료되더라도 지점장이 직접 확인하고 해제해야 합니다. 지점장은 해제 전에 회원과 면담하여 재발 방지 서약을 받는 것을 권장합니다.

| 사유 | 해제 조건 | 방식 |
|---|---|---|
| 미납 | 전액 수납 | 시스템 자동 |
| 자동결제 실패 | 결제 성공 | 시스템 자동 |
| 경미 위반 | 기간 만료 | 시스템 자동 |
| 중대 위반 | 기간 만료 + 지점장 확인 | 수동 |
| 폭력/성범죄 | 블랙리스트 해제 | 본사(회장) |

<Callout type="warning">
폭력·성범죄로 인한 영구 정지는 자동으로 **블랙리스트(A급)**로 전환됩니다. 블랙리스트 해제는 본사 회장 승인과 서약서 제출이 필요하며, 해제 이후에도 재가입 시 지점장 검토를 거칩니다.
</Callout>
