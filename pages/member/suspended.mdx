# 정지(SUSPENDED) 정책

## 정지 사유 분류

| 분류 | 사유 | 기간 | 부과 |
|---|---|---|---|
| 미납 | 미수금 30일 초과 | 수납 시까지 | 시스템 자동 |
| 자동결제 실패 | 3회 연속 실패 | 결제 성공 시까지 | 시스템 자동 |
| 경미 위반 | 시설 규칙 위반 | 7일 | 지점장 |
| 중대 위반 | 시설 파손, 위협 | 14~30일 | 지점장 |
| 폭력/성범죄 | 폭행 등 | 영구 (→ 블랙리스트) | 지점장 + 본사 |

## 정지 중 규칙

| 항목 | 규칙 |
|---|---|
| 출입 | 차단 |
| 예약 | 불가 |
| 결제 | 미납 해소 목적만 |
| 이용 기간 (미납) | 기간 정지 (복귀 시 잔여 유지) |
| 이용 기간 (위반) | 계속 소진 |
| 알림 | 즉시 SMS (사유 + 해제 조건) |

| 사유 | 해제 조건 | 방식 |
|---|---|---|
| 미납 | 전액 수납 | 시스템 자동 |
