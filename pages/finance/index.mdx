# 결제 정책

import { Callout } from 'nextra/components'

결제 수단은 회원의 상황과 상품 유형에 따라 유연하게 선택할 수 있습니다. 각 수단은 서로 다른 특성과 처리 흐름을 가지므로, 담당 스태프는 수단별 규칙을 정확히 숙지해야 합니다. 특히 취소·환불 발생 시 원 결제 수단과 동일한 방식으로 처리해야 하는 원칙을 반드시 지켜야 합니다.

## 할부 결제

할부는 고가의 PT 이용권이나 장기 회원권을 구매할 때 회원의 부담을 분산하는 수단입니다. 신용카드 할부만 허용하며, 무이자 할부는 카드사와의 협약이 필요하므로 반드시 본사 결재를 받은 후 적용해야 합니다. 할부 취소는 카드사 정책에 따라 전월 거래 취소가 불가한 경우가 있으므로, 당일 취소와 과거 취소를 구분하여 권한을 다르게 설정합니다.

| 항목 | 규칙 |
|---|---|
| 할부 수단 | 신용카드만 |
| 개월 수 | 2, 3, 6, 10, 12개월 |
| 무이자 | 본사 결재 후 적용 |
| 최소 금액 | 5만 원+ |
| 처리 권한 | 결제: 스태프+, 취소: 지점장+(당일), 본사(과거) |

## 자동 결제 (정기 결제)

자동 결제는 월 단위로 이용료를 갱신하는 구독형 회원권에 주로 활용됩니다. 회원이 매월 방문해 결제하는 번거로움을 없애고, 센터 입장에서는 안정적인 매출 예측이 가능합니다. 단, 결제 실패가 발생하면 회원에게 즉시 안내하고 재시도 일정을 공지해야 합니다.

**자동 결제 실패 시나리오 예시:**
1월 10일 정기 결제 실패 → 1월 13일(+3일) 1차 재시도 → 실패 시 1월 17일(+7일) 2차 재시도 → 실패 시 1월 24일(+14일) 3차 재시도 → 3회 모두 실패 시 자동결제 해지 및 회원 계정 SUSPENDED 처리. 매 실패마다 SMS 알림이 발송되므로, 회원이 카드 정보를 업데이트할 수 있도록 안내창 또는 앱 링크를 함께 제공합니다.

<Callout type="warning">
자동 결제 등록 시 **회원의 서면 동의**가 반드시 필요합니다. 동의 없이 자동결제를 등록하는 행위는 전자금융거래법 위반입니다.
</Callout>

| 항목 | 규칙 |
|---|---|
| 등록 | 회원 동의 필수, 스태프+ |
| 결제일 | 매월 동일일 |
| 실패 재시도 | 3일 후 → 7일 후 → 14일 후 |
| 3회 실패 | 자동결제 해지 + SUSPENDED |
| 실패 알림 | 매 실패 시 SMS |
| 사전 알림 | 결제일 3일 전 SMS |

## 분납 정책

분납은 고가 상품(PT 패키지 등)을 구매하는 회원이 일시 납부가 어려울 때 지점장 판단으로 허용할 수 있는 예외적 결제 방식입니다. 반드시 분납 약정서를 작성해야 하며, 약정서에는 총 결제 금액, 회차별 납부 금액, 납부 예정일, 미이행 시 서비스 정지 조건, 회원 서명이 포함되어야 합니다.

**미이행 시 프로세스:** 약정일 초과 7일까지는 SMS 안내, 30일을 초과하면 시스템이 자동으로 SUSPENDED 상태로 전환합니다. 정지 해제는 해당 회차 납부 완료 후 지점장이 수동으로 처리합니다. 서비스는 1차 납부 완료 즉시 시작되므로, 2·3차 미납 시에도 이미 이용한 기간에 대한 환불은 적용되지 않습니다.

| 항목 | 규칙 |
|---|---|
| 허용 | 지점장 승인 시 |
| 일정 | 최대 3회, 3개월 내 완납 |
| 미이행 시 | 7일 초과 → SMS, 30일 초과 → SUSPENDED |
| 서비스 시작 | 1차 납부 완료 시 |

## 세금계산서 / 현금영수증

현금 및 계좌이체 결제 시 현금영수증은 자동 발행됩니다. 법인 회원이 세금계산서를 요청하는 경우 지점장 이상이 발행하며, 발행 후 수정이 필요하면 본사(hq_staff 이상)만 수정 발행이 가능합니다. 세법에 따라 모든 증빙 서류는 5년간 보존해야 합니다.

<Callout type="warning">
현금영수증 미발행 또는 거부 시 **소득세법 위반**으로 가산세가 부과될 수 있습니다. 현금 결제 시 반드시 자동 발행 여부를 확인하세요.
</Callout>

| 항목 | 규칙 |
|---|---|
| 현금영수증 | 현금/이체 시 자동 발행 |
| 세금계산서 | 법인 회원 대상, 지점장+ |
| 수정 발행 | 본사(hq_staff+) |
| 보존 | 5년 |

## 복합 결제

복합 결제는 두 가지 이상의 결제 수단을 혼합해 하나의 거래를 처리하는 방식입니다. 예를 들어 100만 원 PT 이용권을 카드 60만 원 + 마일리지 40만 원으로 결제하는 경우가 대표적입니다. 각 수단별 최소 금액(1만 원)을 준수해야 하며, 3가지 이상 수단을 혼합할 경우에는 지점장이 직접 처리해야 합니다.

**환불 시 수단별 처리 순서가 마일리지 → 현금 → 카드인 이유:** 마일리지는 센터 내부 포인트로 즉시 복원이 가능하고, 현금은 계좌이체로 3영업일 내 처리됩니다. 카드 취소는 카드사를 거쳐 3~7영업일이 소요되는 가장 복잡한 수단이므로 마지막에 처리합니다. 이 순서는 회원이 가장 빠르게 실질적인 금전 혜택을 받을 수 있도록 설계된 것입니다.

| 항목 | 규칙 |
|---|---|
| 허용 조합 | 카드+현금, 카드+마일리지, 현금+마일리지 |
| 최소 금액 | 각 수단 1만 원+ |
| 환불 순서 | 마일리지 → 현금 → 카드 |
| 권한 | 스태프+(기본), 3수단+ → 지점장 |

## CRM에서 결제 관리하기

### POS 결제 화면

매출 → POS에서 결제를 처리합니다. 회원을 검색하여 선택한 뒤 상품을 고르고, 결제 수단(카드, 현금, 계좌이체, 마일리지)을 선택하면 결제가 완료됩니다.

- **할부 설정:** 카드 선택 시 할부 개월 수(2, 3, 6, 10, 12개월) 옵션이 표시됩니다. 무이자 할부는 본사 결재 후 적용 가능합니다.
- **분납 등록:** POS 화면에서 "분납" 옵션을 선택하면 회차, 금액, 납부 일정을 설정할 수 있습니다. 등록 완료 후 분납 약정서를 출력하여 회원 서명을 받아야 합니다.

### 자동 결제 등록

회원 상세 → 결제 탭 → "자동결제 등록" 버튼을 클릭합니다. 카드 정보를 입력하고 전자 동의서에 회원 서명을 받으면 자동결제가 등록됩니다.

<Callout type="warning">
자동결제 등록 시 반드시 회원의 서면 동의를 받아야 합니다. 동의 없이 등록하면 전자금융거래법 위반입니다.
</Callout>

### 미수금 관리

매출 → 미수금에서 미납 내역을 조회할 수 있습니다. 기간 및 금액으로 필터링이 가능하며, 해당 건을 선택하고 "수납 처리" 버튼을 클릭해 납부를 기록합니다.

### 매출 현황

대시보드 → 매출 현황에서 일별/월별 매출을 확인할 수 있습니다. 결제 수단별 통계도 함께 제공되며, 우측 상단의 "엑셀 다운로드" 버튼으로 데이터를 내려받을 수 있습니다.
