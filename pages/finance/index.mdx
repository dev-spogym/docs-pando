# 결제 정책

## 할부 결제

| 항목 | 규칙 |
|---|---|
| 할부 수단 | 신용카드만 |
| 개월 수 | 2, 3, 6, 10, 12개월 |
| 무이자 | 본사 결재 후 적용 |
| 최소 금액 | 5만 원+ |
| 처리 권한 | 결제: 스태프+, 취소: 지점장+(당일), 본사(과거) |

## 자동 결제 (정기 결제)

| 항목 | 규칙 |
|---|---|
| 등록 | 회원 동의 필수, 스태프+ |
| 결제일 | 매월 동일일 |
| 실패 재시도 | 3일 후 → 7일 후 → 14일 후 |
| 3회 실패 | 자동결제 해지 + SUSPENDED |
| 실패 알림 | 매 실패 시 SMS |
| 사전 알림 | 결제일 3일 전 SMS |

## 분납 정책

| 항목 | 규칙 |
|---|---|
| 허용 | 지점장 승인 시 |
| 일정 | 최대 3회, 3개월 내 완납 |
| 미이행 시 | 7일 초과 → SMS, 30일 초과 → SUSPENDED |
| 서비스 시작 | 1차 납부 완료 시 |

## 세금계산서 / 현금영수증

| 항목 | 규칙 |
|---|---|
| 현금영수증 | 현금/이체 시 자동 발행 |
| 세금계산서 | 법인 회원 대상, 지점장+ |
| 수정 발행 | 본사(hq_staff+) |
| 보존 | 5년 |

## 복합 결제

| 항목 | 규칙 |
|---|---|
| 허용 조합 | 카드+현금, 카드+마일리지, 현금+마일리지 |
| 최소 금액 | 각 수단 1만 원+ |
| 환불 순서 | 마일리지 → 현금 → 카드 |
| 권한 | 스태프+(기본), 3수단+ → 지점장 |
