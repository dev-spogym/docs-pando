# 환불 / 위약금 정책

## 환불 계산 공식

### 기간제 상품

```
환불 금액 = 결제 금액 - (일일 이용료 × 사용일수) - 위약금
일일 이용료 = 결제 금액 ÷ 총 이용일수
```

### 횟수제 상품

```
환불 금액 = 결제 금액 - (1회 단가 × 사용횟수) - 위약금
```

<Callout type="warning">
1회 단가는 **할인 전 정가** 기준으로 산정 (공정거래위원회 고시)
</Callout>

### 혼합형

```
사용액 = MAX(기간 비례, 횟수 비례)
환불 금액 = 결제 금액 - 사용액 - 위약금
```

## 위약금 정책

> 법적 근거: 소비자분쟁해결기준 (공정거래위원회)

| 조건 | 위약금률 |
|---|---|
| 이용 시작 전 | 0% (전액 환불) |
| 이용 시작 후 | 잔여 금액의 10% |

**위약금 상한:** 잔여 이용 금액의 10% 초과 불가

### 위약금 면제 사유

| 사유 | 서류 |
|---|---|
| 센터 귀책 (폐쇄/이전/축소) | 없음 |
| 계약일 7일 내 청약철회 | 없음 |
| 질병/부상 (3개월+ 이용 불가) | 진단서 |
| 주소 이전 (20km+) | 전입확인서 |
| 해외 이주 | 출국 증빙 |

면제 승인: 지점장 + 본사 결재

## 청약철회 (쿨오프)

<Callout type="error">
**법적 의무:** 어떤 사유로도 거부 불가
</Callout>

| 항목 | 규칙 |
|---|---|
| 가능 기간 | 계약일로부터 7일 이내 |
| 이용 전 | 전액 환불 (위약금 없음) |
| 이용 후 (7일 내) | 결제금액 - 일일이용료×사용일수 |
| 처리 기한 | 3영업일 이내 |
| 처리 권한 | 지점장 |

## 환불 승인 단계

| 금액 | 승인 |
|---|---|
| 50만 원 이하 | 지점장 단독 |
| 50만 원 초과 | 본사 결재 |
| 위약금 면제 건 | 본사 결재 (금액 무관) |

## 환불 수단

| 원 결제 | 환불 수단 | 소요 |
|---|---|---|
| 카드 | 카드 취소 | 3~7영업일 |
| 현금 | 계좌 이체 | 3영업일 |
| 계좌이체 | 원 계좌 이체 | 3영업일 |
| 마일리지 | 마일리지 복원 | 즉시 |
| 복합 | 각 수단별 비례 | 상이 |
