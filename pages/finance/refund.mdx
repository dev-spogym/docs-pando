# 환불 / 위약금 정책

import { Callout } from 'nextra/components'

환불 정책은 소비자 보호 법령을 기반으로 설계되어 있으며, 센터의 자의적 판단으로 법적 기준보다 불리하게 적용할 수 없습니다. 담당자는 환불 요청 시 상품 유형(기간제·횟수제·혼합형)을 먼저 확인하고, 아래 공식에 따라 정확한 금액을 산출해야 합니다.

## 환불 계산 공식

### 기간제 상품

기간제 이용권은 총 이용 가능 일수를 기준으로 일일 이용료를 산정하고, 실제 사용한 일수만큼 차감한 뒤 위약금을 공제합니다.

```
환불 금액 = 결제 금액 - (일일 이용료 × 사용일수) - 위약금
일일 이용료 = 결제 금액 ÷ 총 이용일수
```

**구체적 계산 예시:**
90일 이용권을 100만 원에 구매하고, 30일 사용 후 환불을 요청하는 경우:
- 일일 이용료 = 1,000,000원 ÷ 90일 = 11,111원
- 사용액 = 11,111원 × 30일 = 333,333원
- 잔여 금액 = 1,000,000원 - 333,333원 = 666,667원
- 위약금 = 666,667원 × 10% = 66,667원
- **최종 환불액 = 666,667원 - 66,667원 = 600,000원**

### 횟수제 상품

횟수제(PT 등)는 사용한 횟수만큼의 정가 금액을 차감합니다. 핵심은 1회 단가를 할인 전 정가로 산정한다는 점입니다.

```
환불 금액 = 결제 금액 - (1회 단가 × 사용횟수) - 위약금
```

<Callout type="warning">
1회 단가는 **할인 전 정가** 기준으로 산정합니다. 공정거래위원회 고시에 따른 의무 사항으로, 패키지 할인가로 산정하면 법령 위반입니다.
</Callout>

**구체적 계산 예시:**
PT 30회를 패키지 할인가 150만 원(정가 1회 6만 원)에 구매하고, 10회 사용 후 환불을 요청하는 경우:
- 1회 단가 = 정가 60,000원 (할인가 50,000원이 아님)
- 사용액 = 60,000원 × 10회 = 600,000원
- 잔여 금액 = 1,500,000원 - 600,000원 = 900,000원
- 위약금 = 900,000원 × 10% = 90,000원
- **최종 환불액 = 900,000원 - 90,000원 = 810,000원**

### 혼합형

혼합형은 기간과 횟수 두 조건 중 불리한 쪽(더 많이 사용한 것으로 계산되는 쪽)을 적용합니다. 회원이 기간은 적게 사용했더라도 횟수를 많이 소진했다면 횟수 기준이 적용됩니다.

```
사용액 = MAX(기간 비례, 횟수 비례)
환불 금액 = 결제 금액 - 사용액 - 위약금
```

## 위약금 정책

위약금은 소비자가 계약을 중도 해지함으로써 센터에 발생하는 손실을 보전하기 위한 제도입니다. 공정거래위원회 소비자분쟁해결기준에 따라 잔여 이용 금액의 10%를 초과할 수 없으며, 이용 시작 전 환불은 위약금 없이 전액 환불이 원칙입니다.

> 법적 근거: 소비자분쟁해결기준 (공정거래위원회)

| 조건 | 위약금률 |
|---|---|
| 이용 시작 전 | 0% (전액 환불) |
| 이용 시작 후 | 잔여 금액의 10% |

**위약금 상한:** 잔여 이용 금액의 10% 초과 불가

### 위약금 면제 사유

위약금 면제는 회원의 귀책이 아닌 사유로 이용이 불가한 경우 또는 센터 귀책 사유가 있을 때 적용됩니다. 면제 승인은 지점장 판단과 본사 결재가 모두 필요합니다.

**면제 사유별 실무 처리 절차:**
- **센터 귀책(폐쇄·이전·축소):** 서류 불필요. 지점장이 사유를 시스템에 입력하고 본사 결재 요청 → 대표/회장 승인 후 전액 환불 처리
- **질병·부상:** 진단서(3개월 이상 운동 불가 소견 필수)를 스캔하여 첨부 → 지점장 검토 후 본사 결재 요청
- **주소 이전(20km+):** 전입신고 확인서 또는 주민등록등본 첨부. 직선거리 20km 이상 여부를 지도 서비스로 확인
- **해외 이주:** 출국 증빙(비자, 항공권, 거주증명 등) 첨부

| 사유 | 서류 |
|---|---|
| 센터 귀책 (폐쇄/이전/축소) | 없음 |
| 계약일 7일 내 청약철회 | 없음 |
| 질병/부상 (3개월+ 이용 불가) | 진단서 |
| 주소 이전 (20km+) | 전입확인서 |
| 해외 이주 | 출국 증빙 |

면제 승인: 지점장 + 본사 결재

## 청약철회 (쿨오프)

청약철회는 소비자기본법에 근거한 소비자의 법적 권리입니다. 계약일로부터 7일 이내라면 어떠한 사유로도 거부할 수 없으며, 이를 거부하거나 방해하면 법적 제재를 받을 수 있습니다. 이용을 시작했더라도 7일 이내라면 사용일수에 해당하는 일일 이용료만 차감하고 위약금 없이 환불해야 합니다.

<Callout type="error">
**법적 의무:** 청약철회는 어떤 사유로도 거부 불가합니다. "이미 이용했으니 안 된다"는 거절은 소비자기본법 위반입니다.
</Callout>

| 항목 | 규칙 |
|---|---|
| 가능 기간 | 계약일로부터 7일 이내 |
| 이용 전 | 전액 환불 (위약금 없음) |
| 이용 후 (7일 내) | 결제금액 - 일일이용료×사용일수 |
| 처리 기한 | 3영업일 이내 |
| 처리 권한 | 지점장 |

## 환불 승인 단계

환불 금액에 따라 승인 권한이 달라지는 이유는 고액 환불의 경우 재무적 영향이 크고, 위약금 면제 건은 예외적 결정이기 때문입니다. 50만 원 이하는 지점장이 신속하게 처리하여 회원 불편을 최소화하고, 50만 원 초과 건은 본사의 검토를 통해 적정성을 확보합니다. 위약금 면제는 금액과 무관하게 항상 본사 결재가 필요하며, 이는 과도한 면제 남용을 방지하고 일관된 정책 적용을 보장하기 위함입니다.

| 금액 | 승인 |
|---|---|
| 50만 원 이하 | 지점장 단독 |
| 50만 원 초과 | 본사 결재 |
| 위약금 면제 건 | 본사 결재 (금액 무관) |

## 환불 수단

환불은 원칙적으로 원 결제 수단과 동일한 방식으로 처리합니다. 카드 결제 건을 현금으로 환불하거나, 마일리지 결제 건을 현금으로 환불하는 것은 원칙적으로 허용되지 않습니다. 소요 기간은 카드사 및 은행 처리 일정에 따라 다를 수 있으며, 회원에게 예상 기간을 안내해야 합니다.

| 원 결제 | 환불 수단 | 소요 |
|---|---|---|
| 카드 | 카드 취소 | 3~7영업일 |
| 현금 | 계좌 이체 | 3영업일 |
| 계좌이체 | 원 계좌 이체 | 3영업일 |
| 마일리지 | 마일리지 복원 | 즉시 |
| 복합 | 각 수단별 비례 | 상이 |

## CRM에서 환불 처리하기

### 환불 처리 절차

회원 상세 → 결제 탭에서 환불할 결제 건을 선택하고 "환불" 버튼을 클릭합니다. 환불 계산기가 자동으로 사용일수·횟수와 위약금을 계산하여 최종 환불 금액을 표시합니다. 내용을 확인한 후 승인하면 환불이 진행됩니다.

### 위약금 면제 처리

환불 화면에서 "위약금 면제" 항목을 체크하면 면제 사유 선택 창이 열립니다. 해당 사유를 선택하고 증빙 서류를 첨부하면 본사 결재가 자동으로 생성됩니다. 본사 승인 완료 후 면제가 적용됩니다.

### 청약철회 처리

계약일로부터 7일 이내에 환불을 요청하는 경우, 환불 화면에서 유형이 "청약철회"로 자동 감지됩니다. 위약금 없이 처리되며, 이용한 경우 사용일수에 해당하는 일일 이용료만 차감됩니다.

<Callout type="error">
청약철회는 시스템에서 자동 감지되더라도 처리를 거부하거나 지연할 수 없습니다. 반드시 3영업일 이내에 처리하세요.
</Callout>
