# 매출 마감 정책

## 일 마감

| 항목 | 규칙 |
|---|---|
| 마감 시간 | 영업 종료 시 (기본 23:00) |
| 마감 처리 | 시스템 자동 (일별 매출 snapshot) |
| 당일 수정 | 지점장 |
| 전일 수정 | 본사 결재 |
| 리포트 | 지점장 + 본사 자동 발송 |

## 월 마감

| 항목 | 규칙 |
|---|---|
| 마감일 | 매월 말일 23:59 (자동) |
| 확정 | 본사(대표/회장) — 익월 5일까지 |
| 확정 후 수정 | 회장만 |
| 마감 전 알림 | 말일 3일 전 지점장 알림 |
| 리포트 | 본사 + 각 지점장 자동 배포 |

## 수기 등록 매출

| 항목 | 규칙 |
|---|---|
| 사유 | POS 장애, 현금 직거래, 외부 결제 |
| 권한 | 지점장 |
| 감사 | 본사(대표)에게 실시간 알림 |
| 표시 | AuditLog에 "수기등록" 태그 |
