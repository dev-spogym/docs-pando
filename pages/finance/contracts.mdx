# 계약 / 구독 관리

import { Callout } from 'nextra/components'

계약은 회원이 상품을 구매할 때 체결되는 전자 문서이며, 구독은 월 단위 자동 결제 기반의 정기 이용 서비스입니다.

## 계약 등록 (5단계 위저드)

계약 메뉴 → "새 계약"을 클릭하면 5단계 위저드가 시작됩니다.

### 1단계: 회원 선택
기존 회원 검색 또는 신규 회원 등록 후 선택합니다.

### 2단계: 상품 선택
이용권, PT, GX 등 구매할 상품을 선택합니다. 복수 선택 가능합니다.

### 3단계: 기간/금액 설정
상품별 이용 기간과 결제 금액을 확인하고 조정합니다.

### 4단계: 할인 적용
승인된 프로모션이나 쿠폰을 적용합니다. 할인율은 최대 50%까지 허용됩니다.

### 5단계: 결제
결제 수단을 선택하고 결제를 완료합니다. 계약서 PDF가 자동 생성됩니다.

<Callout type="info">
계약 완료 시 자동으로 처리되는 항목:
- 이용권 생성 (시작일 기준)
- 매출 등록
- 영수증 발행
- 계약서 PDF 저장
</Callout>

## 구독 관리

구독형 상품(월 자동결제 이용권)을 관리하는 화면입니다.

### 구독 현황 대시보드
- **활성 구독 수**: 현재 자동결제 중인 회원 수
- **일시정지**: 회원 요청으로 자동결제를 일시 중지한 수
- **해지**: 구독이 해지된 수
- **MRR(월 반복 매출)**: 구독 기반 월 예상 매출

### 구독 해지
- **즉시 해지**: 해지 즉시 이용 종료
- **월말 해지**: 현재 결제 기간 종료 시 해지

### 결제 실패 처리
자동결제 실패 시 시스템이 자동으로 재시도합니다.
- 1차 재시도: 실패 후 3일
- 2차 재시도: 실패 후 7일
- 3차 재시도: 실패 후 14일
- 3회 모두 실패 시 → 구독 자동 일시정지

매 실패 시 회원에게 SMS 알림이 발송됩니다.

<Callout type="warning">
프로레이타(일할 계산) 환불: 구독 해지 시 남은 기간에 대한 환불은 일할 계산으로 처리됩니다.
</Callout>
