# 지점 운영 정책

import { Callout } from 'nextra/components'

다지점 피트니스 센터 운영에서 가장 어려운 과제 중 하나는 지점별 자율성과 본사의 일관된 통제 사이의 균형을 맞추는 것입니다. 지점마다 상권과 고객층이 달라 획일적 운영이 최적이 아닐 수 있지만, 너무 많은 재량을 허용하면 브랜드 일관성이 무너지고 회원이 지점마다 다른 경험을 하게 됩니다. 이 정책은 일상적인 운영 결정은 지점장에게 위임하고, 브랜드·재무·법적 영향이 큰 결정은 본사 승인을 요구하는 방식으로 균형을 잡습니다.

## 영업 시간

영업 시간은 지역 상권, 유동 인구, 회원 패턴에 따라 지점마다 다를 수 있습니다. 소소한 조정(예: 주말 1시간 단축)은 지점장이 자율적으로 결정하지만, 전체 영업 시간을 크게 줄이거나 정기 휴무일을 도입하는 등 회원 이용 패턴에 영향을 주는 변경은 본사 결재를 거칩니다. 영업 외 시간에는 출입 게이트가 자동으로 잠기고 키오스크가 비활성화되어 무단 출입을 방지합니다.

| 항목 | 규칙 |
|---|---|
| 설정 | 지점장 (소소한 조정), 주요 변경 → 본사 결재 |
| 기본값 | 06:00~23:00 |
| 공휴일/휴무 | 지점장 등록 + 회원 공지 |
| 영업 외 | 출입 차단 + 키오스크 비활성화 |

## 일시중단 (PAUSED)

<Callout type="warning">
지점 일시중단은 **회장 승인**이 필수입니다. 수백 명의 회원 이용이 중단되는 사안이므로, 최고 의사결정권자의 판단을 요구합니다.
</Callout>

지점 일시중단은 시설 보수, 재난·사고, 법적 처분 등의 이유로 영업을 임시 중단하는 상태입니다. 이 기간 동안 회원 보호가 최우선입니다.

**회원 보호 조치:**
- 중단 기간 전체가 이용권 기간에서 차감되지 않고 **자동 연장**됩니다. 예를 들어 3개월 이용권을 보유한 회원이 일시중단 15일을 겪으면, 종료일이 15일 뒤로 자동 밀립니다.
- 중단 7일 전 전체 회원에게 SMS로 공지하여 불편을 최소화합니다.
- 환불을 원하는 회원은 중단 기간 중에도 환불 요청이 가능합니다.
- 타 지점으로의 이관도 허용합니다.

| 항목 | 규칙 |
|---|---|
| 결재 | 회장 승인 필수 |
| 허용 행위 | 조회, 환불, 이관 |
| 차단 행위 | 신규 등록, POS, 수업, 출입 |
| 회원 기간 | 중단 일수만큼 자동 연장 |
| 직원 급여 | 본사 결정 |
| 공지 | 7일 전 전원 SMS |

## 타 지점 이용

타 지점 이용 정책은 회원 편의를 높이는 동시에 지점 간 매출 배분 문제를 내포합니다. 회원이 등록한 지점 외에 다른 지점을 이용하면, 해당 지점의 시설과 서비스를 소비하지만 이용권 매출은 원 지점에 귀속됩니다. 이 때문에 타 지점 이용은 무제한이 아닌 월 4회로 제한하고, 출석 기록에 `isOtherBranch=true` 플래그를 남겨 향후 지점 간 정산이나 분석에 활용합니다. 락커는 지점별 자산이므로 타 지점에서는 사용할 수 없습니다.

| 항목 | 규칙 |
|---|---|
| 허용 | 동일 테넌트 ACTIVE 지점 (본사 설정) |
| 횟수 | 월 4회 |
| 추가 요금 | 없음 (기본), 본사 설정 가능 |
| 출석 기록 | `isOtherBranch=true` |
| 락커 | 타 지점 불가 |

## 상품/가격 차등

가격 차등은 지점의 입지, 시설 수준, 경쟁 환경에 따라 전략적으로 적용하는 프리미엄 전략의 일환입니다. 예를 들어 강남·역삼 같은 프리미엄 상권 지점은 월 회비를 높게 책정하고, 신규 지점은 초기 회원 유치를 위해 낮은 가격으로 시작할 수 있습니다. 단, 가격 결정은 브랜드 전략과 수익성에 직접적인 영향을 미치므로 지점이 임의로 바꿀 수 없으며, 반드시 본사 승인을 거쳐야 합니다. 본사는 템플릿 가격 체계를 관리하면서 지점별 예외를 결재를 통해 허용합니다.

| 항목 | 규칙 |
|---|---|
| 가격 차등 | 허용 (본사 승인) |
| 관리 | 본사 템플릿 + 지점별 override (결재) |
| 통합 조정 | 본사가 전 지점 일괄 가능 |

## 지점 운영 설정 방법

### 영업 시간 설정

설정 메뉴 → 센터 기본정보로 이동하여 요일별 영업 시작·종료 시간을 설정하세요. 변경 사항은 저장 즉시 출입 게이트와 키오스크에 반영됩니다.

### 휴무일 등록

설정 메뉴 → 휴무일에서 "날짜 추가" 버튼으로 특정 날짜를 휴무일로 등록할 수 있습니다. 휴무일이 등록되면 해당 날짜의 이용권은 자동으로 연장되며, 회원에게 공지 SMS가 발송됩니다.

### 지점 상태 변경

본사 역할로 로그인한 후 지점 관리 화면의 지점 목록에서 해당 지점을 선택하면 상태 변경 버튼이 표시됩니다. PAUSED 또는 CLOSING으로 변경하려면 회장 승인이 필요합니다.

### 타 지점 이용 기록 확인

출석 현황 화면에서 필터를 열고 "타 지점 방문" 항목을 선택하면 `isOtherBranch` 플래그가 표시된 출석 내역만 조회됩니다. 월별 타 지점 이용 횟수가 4회를 초과한 회원은 별도 안내가 필요합니다.
