# 지점 운영 정책

## 영업 시간

| 항목 | 규칙 |
|---|---|
| 설정 | 지점장 (소소한 조정), 주요 변경 → 본사 결재 |
| 기본값 | 06:00~23:00 |
| 공휴일/휴무 | 지점장 등록 + 회원 공지 |
| 영업 외 | 출입 차단 + 키오스크 비활성화 |

## 일시중단 (PAUSED)

| 항목 | 규칙 |
|---|---|
| 결재 | 회장 승인 필수 |
| 허용 행위 | 조회, 환불, 이관 |
| 차단 행위 | 신규 등록, POS, 수업, 출입 |
| 회원 기간 | 중단 일수만큼 자동 연장 |
| 직원 급여 | 본사 결정 |
| 공지 | 7일 전 전원 SMS |

## 타 지점 이용

| 항목 | 규칙 |
|---|---|
| 허용 | 동일 테넌트 ACTIVE 지점 (본사 설정) |
| 횟수 | 월 4회 |
| 추가 요금 | 없음 (기본), 본사 설정 가능 |
| 출석 기록 | `isOtherBranch=true` |
| 락커 | 타 지점 불가 |

## 상품/가격 차등

| 항목 | 규칙 |
|---|---|
| 가격 차등 | 허용 (본사 승인) |
| 관리 | 본사 템플릿 + 지점별 override (결재) |
| 통합 조정 | 본사가 전 지점 일괄 가능 |
