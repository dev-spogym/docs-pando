# 할인 / 프로모션 정책

import { Callout } from 'nextra/components'

할인 정책은 회원 유치와 매출 보호 사이의 균형을 맞추는 핵심 도구입니다. 할인 권한이 무제한으로 열려 있으면 현장 직원의 임의 할인으로 인해 매출이 유실되고, 지점 간 가격 일관성이 무너집니다. 반면 할인이 전혀 없으면 경쟁 센터 대비 영업력이 떨어집니다. 이 정책은 역할별 할인 상한을 두어 영업 유연성을 허용하되, 일정 수준 이상은 본사 승인을 통해 통제합니다.

## 할인율 상한

역할에 따라 적용할 수 있는 최대 할인율이 다르며, 상한을 초과하려면 상위 결재가 필요합니다. 이 구조는 현장의 영업 판단을 존중하면서도 회사 전체의 수익성을 보호하기 위한 장치입니다.

| 역할 | 최대 할인율 | 초과 시 |
|---|---|---|
| 스태프 / FC | 사전 설정 정책만 | - |
| 지점장 | 10% | 📋 본사 결재 |
| 본사 직원 | 20% | 대표/회장 승인 |
| 대표 | 30% | 회장 승인 |
| 회장 | 제한 없음 | - |

- 50% 이상: 회장 필수 + AuditLog 강화
- 100% (무료): 회장만

<Callout type="warning">
50% 이상 할인은 감사 로그에 강화 기록이 남습니다. 회원 계정, 처리 직원, 사유, 승인자가 모두 기록되며 이후 감사 시 열람 대상이 됩니다.
</Callout>

## 쿠폰/프로모션 생성

<Callout type="info">
지점에서 쿠폰이나 프로모션을 만들려면 **반드시 본사 결재**를 거쳐야 합니다.
</Callout>

지점에서 독자적으로 쿠폰을 만들 수 없는 이유는, 쿠폰 남발이 브랜드 가치를 훼손하고 지점 간 불균형을 초래하기 때문입니다. 본사 결재를 통해 프로모션의 유효성, 예산 여력, 타 지점과의 형평성을 함께 검토합니다.

**실제 시나리오 예시:**

강남점 지점장이 여름 프로모션으로 "3개월 등록 시 20% 할인" 쿠폰을 기획한다면, 아래 절차를 따릅니다:

1. 지점장이 CRM에서 결재 요청 작성 (할인율, 대상 상품, 유효기간, 발급 수량 포함)
2. 본사 직원이 검토 후 의견 첨부 (예산 영향, 타 지점 형평성 등)
3. 대표 또는 회장이 최종 승인/반려
4. 승인 완료 후 시스템에 쿠폰 자동 등록
5. 강남점 지점장·FC·스태프가 승인된 쿠폰을 회원에게 자유롭게 발급

```
지점장: 기획 → 본사 결재 요청 → 대표/회장 승인
→ 시스템 등록 → 지점에서 승인된 쿠폰 자유 발급
```

## 중복 할인

중복 할인 제한은 과도한 할인 적용을 방지하기 위한 비즈니스 로직입니다. 프로모션과 쿠폰은 동일한 목적(신규 유치·재등록 유도)으로 설계되므로 중복 적용을 허용하면 예상치 못한 수준의 매출 손실이 발생합니다. 반면 마일리지는 회원이 이미 적립한 포인트를 소진하는 방식이므로 할인과 중복 사용이 가능합니다.

| 항목 | 규칙 |
|---|---|
| 프로모션 + 쿠폰 | 중복 불가 (큰 것만) |
| 할인 + 마일리지 | 중복 가능 |
| 최대 할인 상한 | 결제 금액의 30% |
| 마일리지 포함 상한 | 결제 금액의 50% |

## 제휴/법인 할인

제휴·법인 할인은 B2B 영업의 핵심 수단입니다. 인근 기업이나 기관의 임직원을 대상으로 단체 등록을 유치하면, 개별 영업보다 훨씬 효율적으로 회원 수를 늘릴 수 있습니다. 법인 고객은 세금계산서 발행이 필요하므로 일반 소비자 결제와 처리 방식이 다릅니다.

모든 제휴 계약은 본사가 관리하여 할인율·계약 기간·조건의 일관성을 유지하고, 지점 단위의 임의 제휴로 인한 계약 분쟁을 예방합니다.

| 항목 | 규칙 |
|---|---|
| 등록 | 본사 결재 |
| 할인율 | 업체별 10~20% |
| 법인 결제 | 세금계산서 발행 |
| 계약 관리 | 본사 통합 |

## 소개/추천

소개·추천 프로그램은 기존 회원을 자발적 영업 채널로 활용하는 가장 비용 효율적인 회원 유치 방법입니다. 만족한 회원이 지인을 추천하면, 광고비 없이 신규 회원을 확보하면서 동시에 기존 회원의 만족도와 소속감도 높아집니다. 피추천인에게는 등록비 면제 또는 이용 기간 연장 혜택을 주어 등록 결정을 낮추고, 추천인에게는 마일리지를 지급하여 재참여 동기를 부여합니다. FC에게도 유치 인센티브가 귀속되어 현장 영업 활동을 촉진합니다.

| 항목 | 규칙 |
|---|---|
| 추천인 혜택 | 마일리지 30,000P |
| 피추천인 혜택 | 등록비 면제 또는 +7일 |
| 보상 시점 | 결제 완료 후 즉시 |
| 실적 | FC에게 유치 인센티브 |

## 할인/쿠폰/마일리지 적용 방법

### POS에서 할인 적용

POS 결제 화면에서 결제 금액 입력 후 "할인 적용" 버튼을 누릅니다. 본사에서 승인된 프로모션 목록이 표시되면 해당 항목을 선택하거나, 지점장 권한으로 수동 할인율을 직접 입력할 수 있습니다.

### 쿠폰 발급

본사에서 승인된 쿠폰은 지점장, 스태프, FC가 자유롭게 발급할 수 있습니다. 쿠폰 메뉴에서 발급할 쿠폰을 선택한 후 회원을 검색하여 개별 발급합니다. 동일 조건의 쿠폰을 여러 회원에게 한꺼번에 발급할 때는 "일괄 발급" 기능을 사용하세요.

<Callout type="info">
승인되지 않은 쿠폰은 발급 목록에 표시되지 않습니다. 새 쿠폰이 필요하면 본사 결재를 먼저 요청하세요.
</Callout>

### POS에서 마일리지 차감 결제

POS 결제 화면에서 회원을 조회하면 보유 마일리지가 표시됩니다. "마일리지 사용" 항목에 차감할 포인트를 입력하면 해당 금액만큼 결제 금액에서 차감됩니다. 마일리지와 할인은 중복 사용이 가능하나 마일리지 포함 최대 할인 한도(50%)를 초과할 수 없습니다.

### 마일리지 수동 적립 / 차감

회원 상세 화면의 "마일리지" 탭에서 포인트를 수동으로 적립하거나 차감할 수 있습니다. 적립 또는 차감 버튼을 누른 후 금액과 사유를 입력하고 저장합니다. 수동 처리 내역은 마일리지 이력에 모두 기록됩니다.
