# 할인 / 프로모션 정책

import { Callout } from 'nextra/components'

## 할인율 상한

| 역할 | 최대 할인율 | 초과 시 |
|---|---|---|
| 스태프 / FC | 사전 설정 정책만 | - |
| 지점장 | 10% | 📋 본사 결재 |
| 본사 직원 | 20% | 대표/회장 승인 |
| 대표 | 30% | 회장 승인 |
| 회장 | 제한 없음 | - |

- 50% 이상: 회장 필수 + AuditLog 강화
- 100% (무료): 회장만

## 쿠폰/프로모션 생성

<Callout type="info">
지점에서 쿠폰이나 프로모션을 만들려면 **반드시 본사 결재**를 거쳐야 합니다.
</Callout>

```
지점장: 기획 → 본사 결재 요청 → 대표/회장 승인
→ 시스템 등록 → 지점에서 승인된 쿠폰 자유 발급
```

## 중복 할인

| 항목 | 규칙 |
|---|---|
| 프로모션 + 쿠폰 | 중복 불가 (큰 것만) |
| 할인 + 마일리지 | 중복 가능 |
| 최대 할인 상한 | 결제 금액의 30% |
| 마일리지 포함 상한 | 결제 금액의 50% |

## 제휴/법인 할인

| 항목 | 규칙 |
|---|---|
| 등록 | 본사 결재 |
| 할인율 | 업체별 10~20% |
| 법인 결제 | 세금계산서 발행 |
| 계약 관리 | 본사 통합 |

## 소개/추천

| 항목 | 규칙 |
|---|---|
| 추천인 혜택 | 마일리지 30,000P |
| 피추천인 혜택 | 등록비 면제 또는 +7일 |
| 보상 시점 | 결제 완료 후 즉시 |
| 실적 | FC에게 유치 인센티브 |
