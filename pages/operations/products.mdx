# 상품 관리

import { Callout } from 'nextra/components'

상품은 센터가 회원에게 판매하는 모든 서비스의 기본 단위입니다. 이용권(Membership), PT, GX, 부가상품(락커, 운동복 등) 등 4개 카테고리로 구분되며, 각 상품의 가격·기간·횟수 설정에 따라 회원의 이용 조건이 결정됩니다.

<Callout type="info">
상품 등록/수정/삭제는 **본사 결재가 필수**입니다. 지점에서 임의로 상품을 변경할 수 없습니다.
</Callout>

## 상품 목록

상품 메뉴에서 전체 상품을 확인할 수 있습니다.

- **카테고리 필터**: 이용권, PT, GX, 부가상품 중 선택
- **상태 필터**: 활성/비활성 구분
- **키오스크 노출 필터**: 키오스크에 표시할 상품만 필터링
- 각 상품을 클릭하면 상세 정보를 확인하고 수정할 수 있습니다

| 표시 항목 | 설명 |
|---|---|
| 상품명 | 상품 이름 |
| 카테고리 | 이용권/PT/GX/부가상품 |
| 현금가 | 현금 결제 시 가격 |
| 카드가 | 카드 결제 시 가격 |
| 기간 | 이용 가능 기간 (일) |
| 상태 | 활성/비활성 |

## 상품 등록

상품 목록 → "새 상품" 버튼을 클릭하면 등록 화면으로 이동합니다.

### 공통 입력 항목
- **상품명**: 동일 카테고리 내 중복 불가
- **카테고리**: 이용권, PT, GX, 부가상품 중 선택
- **현금가 / 카드가**: 결제 수단별 가격 (다를 수 있음)
- **상태**: 활성/비활성
- **키오스크 노출**: ON/OFF

### 카테고리별 추가 항목

| 항목 | 이용권 | PT | GX | 부가상품 |
|---|:---:|:---:|:---:|:---:|
| 기간 (일) | ✅ | ✅ | ✅ | ✅ |
| 횟수 | - | ✅ | ✅ | - |
| 1회 정가 | - | ✅ | - | - |
| 정원 | - | - | ✅ | - |
| 자동결제 | ✅ | - | - | - |

<Callout type="warning">
PT 상품의 **1회 정가**는 환불 시 사용액 계산의 기준이 됩니다. 패키지 할인가가 아닌 정가로 설정해야 합니다.
</Callout>

## 상품 수정

상품 목록에서 상품 클릭 → 수정 화면으로 이동합니다. 가격 변경 시 기존 회원의 이용권에는 영향 없이, 신규 구매 건부터 적용됩니다.

## 상품 비활성화

판매를 중단할 상품은 "비활성화"로 전환합니다. 비활성화된 상품은 POS와 키오스크에서 노출되지 않지만, 기존 회원의 이용권은 유지됩니다.
