# 시설 운영 정책

## 락커 만료/갱신

| 항목 | 규칙 |
|---|---|
| 만료 처리 | 매일 00:00 배치 → AVAILABLE |
| 사전 알림 | 7일, 3일, 1일 전 SMS |
| 미수거 | 만료 후 7일 보관 → 14일 후 별도 보관 → 30일 후 폐기 |
| 강제 반납 | 지점장 |

## 출입 통제

| 회원 상태 | 출입 |
|---|---|
| ACTIVE | ✅ |
| EXPIRED/SUSPENDED/HOLDING | ❌ |
| 미납 30일 미만 | ✅ (알림만) |
| 미납 30일+ | ❌ |
| 체험 | ✅ (당일 한정) |
| 비회원 동반 | ✅ (스태프 수동, 기록) |

## 장비/기구 관리

| 항목 | 규칙 |
|---|---|
| 자산 목록 | 지점장 |
| 정기 점검 | 월 1회 |
| 고장 신고 | 스태프+ |
| 수리 | 지점장 (100만원+ → 본사 결재) |
| 폐기 | 지점장 (50만원+ → 본사 결재) |

## 운동복 분실/파손

| 상황 | 처리 |
|---|---|
| 미반납 3일+ | 대여료 3배 과금 |
| 분실 | 정가 변상 |
| 파손 | 수선비 또는 정가 |
